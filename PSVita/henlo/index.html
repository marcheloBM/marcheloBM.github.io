Instrucciones de actualización manual

  1. Abra la aplicación Configuración

  2. Seleccione "Actualización del sistema" y luego "Actualizar mediante Wi-Fi".

  3. Deberías ver una ventana emergente que dice que hay una nueva actualización.. Asegúrese de que la nueva versión de actualización sea "3.74"! Si la versión es diferente, no proceder.

  4. Después de asegurarse de que está actualizando a 3.74, continúe con los pasos que aparecen en pantalla para actualizar.

  5. Después de actualizar a 3.74, puedes volver a visitar este sitio web para hacer jailbreak a tu consola..

¿Qué es HENlo?

HENlo es el quinto jailbreak público para PlayStation Vita y PlayStation TV. Es similar a hacer jailbreak a tu iPhone o rootear tu dispositivo Android. Le permite realizar modificaciones en el kernel y en el usuario, cambiar la velocidad del reloj, instalar complementos, ejecutar homebrews y mucho más.


¿Qué le hará a mi dispositivo?

Cuando haces clic en "desbloquear", HENlo explotará esta consola y cargará el programa de arranque henlo_jb que luego podrás usar para iniciar el marco taihenkaku e instalar la caja de herramientas VitaDeploy..
