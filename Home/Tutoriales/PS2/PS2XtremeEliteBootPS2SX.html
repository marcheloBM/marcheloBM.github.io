PS2 XtremeEliteBoot + PS2SX

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32
  2. Una Computadora para las APP

Programas para este Tutorial

  1. XtremeEliteBoot
  2. PS2SX

Empezando este Tutorial

  1. Preparando nuestra USB
    • Descargamos el archivo XtremeEliteBoot y lo descomprimimos y lo pasamos a la raíz de nuestro USB (?:XEBPLUS)
    • Para ejecutar la aplicación vamos a localizar el archivo dentro de la carpeta XEBPLUS (?:XEBPLUS)
    • El archivo a ejecutar en nuestra PS2 es XEBPLUS_XMAS.ELF una vez localizado podemos colocar nuestra USB en nuestra consola
  2. Una vez en nuestra consola PS2
    • Una vez terminado con la descargar y configuraciones de todo los que necesitamos proseguimos con las configuraciones en la consola PS2
    • Nos vamos a configuraciones de Free MC Boot o FunTuna
      1. Opción 1 (para dejarlo en OPL cuando arranque)
        • Nos vamos a las configuraciones de del OPL
        • Una ves dentro nos vamos a Configura E1 lanunch keys…
        • Buscamos el que dice Auto: y si tiene alguna ruta configurada le damos a para limpiar
        • Y colocamos la nueva ruta ej (Auto: MASS:/XEBPLUS/XEBPLUS_XMAS.ELF).
        • Y listo
      2. Opción 2 (para dejarlo en el menú principal)
        • Nos vamos a las configuraciones de del OPL
        • Buscamos Configure OSDSYS options…
        • Después en Configure Item 1 : y buscamos un ítem vacío para colocarlo
        • Lo seleccionamos y dentro colocamos el
          1. Name (nombre): XtremeEliteBoot
          2. Path1: (ruta del ejecutable): MASS:/XEBPLUS/XEBPLUS_XMAS.ELF
        • Y listo
      3. Una vez terminado los cambios
        • Le damos a Save CNF to MC0 con esto estaríamos guadando los cambios en la memory Card 1
        • Esperamos el mensaje de y le damos a Exit, Return to FunTuna o apagamos y encendemos de nuevo la consola
        • Y listo

Para Comenzar este Tutorial de PS1 en USB

  1. Preparando nuestro Juegos de PS1 en USB
    • Descargamos el programa PS2SX-v1.3e y lo descomprimimos en nuestro PC
    • Para ejecutar la aplicación buscamos dentro de la carpeta extraída PS2SX-v1.3e\PC\ Howling Wolf & Chelsea's PS2SX [1.3E-??].exe donde las ?? significa el idioma de nuestro programa EN=Ingles y ES=Español
    • Lo ejecutamos como administrador para no tener problemas de compatibilidad
    • Una vez abierto nuestra aplicación
      1. Podemos cambiar la ubicación de trabajo de la APP o la dejamos por defecto (tener en cuenta la elección que elijamos) lo demás lo dejamos como esta
      2. Le damos a Preparar para PSX, instalara y crear los archivos necesarios para ejecutar la APP, son 2 carpetas (HWC-PSX y PS2SX)
      3. Para agregar juegos le damos a Adicionar Juegos y buscamos nuestros juegos en formato (*.img, *.bin, *.iso, *.mdf)
      4. Seleccionamos el nombre y Configuración y listo podemos seguir agregando más juegos
      5. Podemos cambiar los cover manual mente
      6. Para terminar, le damos a Terminar Instalación y podemos copiar las carpetas
      7. La carpeta es (HWC-PSX) la podemos copiar a nuestra USB a la raíz (?:)
      8. Y estaríamos listo, solo a probar en nuestra consola

Empezando este Tutorial

  • Para cambiar los temas en esta aplicación
    1. Buscamos en la ruta (?:\XEBPLUS\THM)
      • Seleccionamos el tema que deseamos cambiar por defecto el tema que vienes es Default
      • Copiamos el nombre de la carpeta del tema que deseamos implementar
  • Buscamos en la siguiente ruta (?:XEBPLUS/CFG)
    1. Abrimos el archivo xebplus.cfg
    2. Y buscamos XEBPlusTheme="??????" donde ??... colocaremos el nombre de la carpeta copia con el tema ej: XEBPlusTheme="XMBWave"
    3. Guardamos el archivo con los cambios
    4. Y listo
