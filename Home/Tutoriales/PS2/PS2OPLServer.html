PS2 OPL Server

Para esta Guia

Requisitos para este tutorial

  1. Un sistema Windows 7 o superior

Programas para este Tutorial

  1. OPL Server

Para implementar este tutorial

  1. Nos vamos a panel de control de nuestra computadora
    • Dentro del panel de control buscamos
    • Sistemas y seguridad/firewall de Windows Defender
    • Y en el menú nos vamos a Configuración avanzada
    • En Reglas de entrada en acciones le damos a nueva regla
      1. Le asignamos Puerto siguiente
      2. En puerto locales especifico: 1024
      3. Le damos a siguiente/
      4. Acción le damos a permitir la conexión siguiente
      5. En Perfil siguiente
      6. En nombre le asignamos en nombre PS2Entrada
      7. Y en descripción le asignamos -Entrada PS2
      8. Y finalizar
    • En programas y características
      1. Nos vamos a Activar o desactivar las características de Windows
      2. Dentro vamos a buscar Compatibilidad con el protocolo para compartir Archivos SMB 1.0/CIFS
      3. Y seleccionamos Servidor SMB 1.0/CIFS
      4. Y le damos aceptar
      5. Buscará y Liberia para los archivos necesarios para esta característica
      6. Podemos reiniciar el ordenador para completar bien la instalación
    • Para continuar debemos descomprimir el archivo OPL-Server…
      1. Por primera vez debemos compartir la carpeta PS2
        • Para esto demos compartir la carpeta le damos a clip derecho sobre la carpeta PS2 y en propiedades
        • En la pestaña de Compartir le damos a y le damos en Compartir… y seleccionamos todos y agregamos
        • Le damos los permisos de Lectura y escritura y al botón Compartir y listo
        • Y en uso compartido avanzado le damos al botón
        • Le damos a compartir esta carpeta y en los permisos le damos a Todos con control total y aceptar y aceptar y cerramos la ventana
      2. para agregar juegos los pasamos a la carpeta PS2
        • sí están en ISO los colocamos en la carpeta llamada DVD dentro de PS2
        • si queremos pasarle las caratulas y dejarlo listo
        • abrimos OPL Manager.exe como administrador
        • se queremos seleccionar una ruta y no nos aparece la opción
          1. vamos a Ajustes/cambiar Modo (carpeta OPL)
          2. y e ruta seleccionamos la carpeta PS2 recién creada y compartida
          3. y creamos las carpetas faltantes para automatizar el programa
        • una vez listo nuestro programa nos detectara nuestros juegos
          1. solo nos quedaría corregir el nombre con el botón intentar actualizar el nombre del juego
          2. sí nos da un error debemos reducir el nombre a menos de 32 caracteres
          3. y estaríamos listo
        • Para descargar la caratulas vos vamos a Operaciones/Descargar Imágenes
          1. En la ventana seleccionamos todo lo queremos descargar en Opciones de descarga
          2. Y le damos a comenzar y una vez terminado le damos aceptar y cerramos la ventana
          3. Y estaría todo listo y podemos cerrar todo el programa
      3. Dentro de la carpeta buscamos el archivo OPLServer.exe y lo abrimos como administrador
      4. Le damos los permisos necesarios para ejecutar nuestra aplicación
      5. Para arrancar nuestro servidor le damos a Server is stopped (press to start)
  2. En nuestra PS2 nos vamos al Open PS2 Loader
    • Muy importante que nuestra consola PS2 este conectada a la misma RED de nuestro ordenador (PC o Notebook)
    • Nos vamos a configuración de OPL en Settings (Configuración principal)
      1. Nos vamos a ETH Prefix Path (Ruta predeterminada conexión de red) en not set (sin definir)
      2. Y en ETH Device Star Mode (Modo de inicio para RED) en Auto (Automatico)
      3. Y le damos en aceptar y nos daría Error para eso
    • Nos vamos a Network Settings (Conexión de red)
      1. En advanced options (Opciones avanzadas) lo dejamos en On (activado)
      2. Y en addres Type (Tipo de dirección) lo dejamos en IP
        • Y le asígnanos la ip de nuestro PC o notebook para esto en nuestro equipo colocamos el comando CMD en inicio y abrimos la terminal y colocamos el siguiente comando ipconfig
        • Y buscamos la ip del dispositivo (Wifi o Ethernet) dependiendo de cómo esté conectado nuestro equipo
      3. Y en port(puerto) lo dejamos en en 1024
      4. Y en Share (Unidad o carpeta compartida) colocaremos el nombre de la carpeta compartida PS2
      5. Y listo le damos a aceptar con
    • Y le damos a Save Changes (Guardar cambios)
  3. Y nos vamos hacia tras con para ir a SMB
    • Si todo salió bien nos mostrar nuestros juegos recién cargados en nuestra PS2
    • Y podemos abrir para probar nuestros juegos y estaríamos listo
  4. Recomendaciones para que esto funciones
    • Tiene estar nuestro equipo en la misma RED y todas las configuraciones necesarias
    • Puede cambiar la IP de nuestro Equipo cada 7 días esto se debe al Moden para esto hay q seleccionar la nueva IP asignada de nuestro moden
  5. Y estaría lista para poder jugar nuestros juegos
