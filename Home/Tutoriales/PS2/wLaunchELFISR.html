Requisitos para este tutorial

  1. USB en formato FAT32

Programas para este Tutorial

  1. FunTuna
  2. WEB Oficial

Empezando este Tutorial

  1. Descargamos la última versión disponible de wLaunchELF_ISR y la descomprimimos
  2. Copiamos los archivos a nuestra USB
  3. Una vez copiado nos vamos a nuestra PS2 y abrimos la aplicación launchELF
  4. Y buscamos la carpeta copia y el archivo BOOT-EXFAT.elf ruta MASS:/ y lo abrimos
  5. Y listo
