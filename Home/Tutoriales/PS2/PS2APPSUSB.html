PS2 APPS en USB

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32
  2. Un sistema Windows 7 o superior

Programas para este Tutorial

  1. APPS En USB
  2. Pack de Apps

Para Empezar este Tutorial

  1. Conectamos nuestro USB a nuestro equipo o nuestra ruta de nuestro OPL Server
    • Y buscamos nuestra carpeta APPS para poder pasarle nuestras aplicaciones o emulador a nuestra PS2
    • Si no se encuentra la podemos crear de la siguiente forma crear carpeta con el nombre de APPS
    • Una vez listo nuestra carpeta
  2. Para poder automatizar esto tenemos 3 formas
    • La forma manual para automatizar
      1. Creamos un nuevo documento de texto
      2. Lo renombramos con el siguiente nombre y extensión conf_apps.cfg
      3. Si no podemos editarlo podemos usar el boc de notas o el programa Notepad++
      4. Una vez dentro lo editamos de la siguiente forma para nuestra Aplicaciones
      5. Colocamos el nombre de la aplicación y después la ruta del archivo .elf ej: OPL v0.8=mass:/APPS/ OPL 0.8.ELF o con carpeta RETROARCH=mass:/APPS/retroarch/raboot.elf y así en cada una de las aplicaciones que emos pasado
        • Si están en nuestra aplicación en la memory card cambiamos el mass por mc0 si en la memory 1 y la forma automática por si esta en la 1 o 2 seria ms?
      6. Guardamos los cambios de nuestro archivo .cfg
    • La forma Automática con el archivo .bat
      1. Pasamos el archivo conf_apps.cfg a nuestra carpeta APPS
      2. Seleccionamos la forma de uso si es por USB seleccionamos 1, si es por OPL Server 2
      3. Una vez seleccionado nuestra opción le damos a enter
      4. Y esperamos a que procese los archivos una ves termina le damos a cualquier tecla para terminar
      5. Se nos creará una carpeta con el nombre de cfg dentro estará nuestro archivo conf_apps.cfg
    • Método por carpeta, para las versiones oficial de la v.12.77 y superior (Método no probado)
      1. Copiamos el archivo title.cfg_maker.BAT a nuestra carpeta APPS y lo abrimos
      2. Creará una carpeta por cada aplicación .elf que pasamos a nuestro USB p OPL Server
      3. Una vez terminada
    • Una vez creado el archivo lo colocamos en
      1. Si es USB lo dejamos en la raíz del USB
      2. Si es Server lo dejamos en la raíz del server
      3. Si no podemos acceder iniciamos uLaunchELf y vamos a mass:/ copiamos el archivo conf_apps.cfg con el Copy y lo pegamos en mc0:/OPL/ con paste y reiniciamos la consola
  3. En nuestro OPL hay un menú de Apps para nuestras aplicaciones
    • a. Para acceder a este menú solo de damos a esta posicionarnos en el menú APPS
    • b. Y estaría listo
