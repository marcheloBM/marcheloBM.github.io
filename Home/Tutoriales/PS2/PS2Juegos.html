PS2 Añadir Juegos

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32

Programas para este Tutorial

  1. USBUTIL
  2. OPL Manager

Para Empezar este Tutorial

  1. Para cargar Juegos rápido sin la necedad de caratulas o información del juego solo las ISO
    • Las podemos pasar a nuestro USB en la carpeta llamada DVD
    • Si no está creada la creamos con el mismo nombre
    • El problema es que no podemos pasar juego mayor de 4GB ya que el formato de nuestro USB no nos permite, pero si podemos pasar cualquier juego menor a 4GB
  2. La otra opción para añadir juegos más completa
    • Para esto descargaremos y descomprimir el archivo USBUTIL.rar (si no tiene el programa puede descárgalo desde Nuestra WEB)
    • Para ejecutar nuestro programa buscamos entro de la carpeta el ejecutable USBUtil v2.2 rev1.0.exe
    • Una vez abierto el programa por primera vez
      1. Le damos a continuar y en el menú de arriba
      2. Vamos a archivo y le damos a crear juego desde ISO
      3. Buscaremos el origen la carpeta donde están alojada nuestras ISO de nuestro juego de PS2
      4. Podemos seleccionar una o todas las ISO para pasar a nuestro USB
      5. En destino seleccionamos nuestra USB con la letra
      6. Si queremos podemos cambiar el nombre del juego en Nombre para el nuevo juego (no más de 32 caracteres mientras más corto mejor)
      7. Lo de medio lo dejamos en AUTO y Crear y listo
      8. Esperamos que termine y una vez terminado le damos a Cerrar
      9. Y nos mostrar la lista con los juegos recién pasados
      10. Podemos Renombrar o Eliminar nuestro juego con los botones de abajo
      11. Si queremos añadir más juegos podemos ir a Archivo/Crear juego desde ISO y hacemos lo mismo
      12. Y listo.
    • Si le queremos agregar más juegos a la lista ya creada
      1. Le damos a abrir juegos ul.cfg
      2. Buscaremos nuestro destino en directorios con la letra asignada
      3. Y seleccionamos el archivo ul.cfg creado con anterioridad y le damos a cargar
      4. Y nos cargara nuestra lista de juegos creada
      5. Si queremos añadir más juegos podemos ir a Archivo/Crear juego desde ISO
      6. Y podemos añadir más juegos
  3. Para añadir caratulas o corregir nombre
    • Para esto abriremos nuestra APP OPL Manager
      1. Si es la primera vez que se abre nuestra aplicación nos preguntara el Idioma de preferencia seleccionamos el Idioma
      2. En ruta le damos la ruta de nuestro USB con la letra asignada
      3. Si no estén las carpetas las creara automáticamente
      4. Una vez listo nos detectara nuestro juego en la USB
      5. Le asignamos lo que queremos descargar (caratulas, discos o logotipo) y le damos a comenzar
      6. Una vez terminado podemos cerrar la ventana y estaría listo las caratulas y la información
      7. Si queremos cambiar o agregar más información le damos a Editar CFG y podemos agregar más Información
      8. Y estaría listo
  4. Una vez terminado todo podemos conectar nuestra USB en la consola de PS2
    • Y abrir OPL y esperamos que cargue nuestra información recién cargada
    • En las configuraciones del OPL revisamos las siguientes configuraciones
      1. Si lo tenemos en ingles seria lo siguiente
      2. Pero si lo tenemos en ingles seria
      3. USB Device Start Mode en Auto
      4. En Default Menu en USB Games y le damos a OK
      5. Si queremos en Display Settings le podemos dejar el Widescreen en On
      6. Para ver los cover le damos a Enable Cover Art en On
      7. Cuando términos le damos a Save Changes para guardar los cambios
    • Si lo tenemos en español seria lo siguiente
      1. En configuraciones principal
      2. Logotipo Playstation 2 en activado
      3. Modo de inicio para BDM en Automático
      4. Modo de inicio para Aplicaciones Automático
      5. Menú predeterminado en Juegos BDM y volvemos al menú anterior
      6. En pantalla
      7. Caratulas, serigrafias y fondos en activado
      8. Pantalla panorámica solo probar para ver cual se adapta mejor a sus conexiones
      9. Y listo cuando terminemos le damos a guardar los cambios
    • Una vez terminado le damos a Games List con
  5. Una vez listo se cargarán las caratulas de nuestros juegos podremos jugar nuestros juegos recién pasados a nuestro USB.
