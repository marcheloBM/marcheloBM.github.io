PS2Temps

Aquí hay un ELF (aplicación de krat0s) para mostrar las temperaturas de tu PS2. Funciona con SCPH-500XX y modelos posteriores.

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32

Archivos Para descargar

Archivos Servidores
PS2Temps Mega WEB Oficial

Empezando este Tutorial

  1. Descargamos la última versión disponible de PS2Temps y los demás para probarlos y la descomprimimos
  2. Copiamos los archivos a nuestra USB
  3. Una vez copiado nos vamos a nuestra PS2 y abrimos la aplicación launchELF
  4. Y buscamos la carpeta copia y el archivo PS2Temps.elf ruta MASS:/ y lo abrimos
  5. Una vez dentro de la aplicación nos mostrara la tempera de nuestra consola
  6. Y listo
