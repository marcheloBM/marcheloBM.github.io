PS2 Temas OPL

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32
  2. Un sistema Windows 7 o superior

Programas para este Tutorial

  1. Pack de TemasOPL

Para Empezar este Tutorial

  1. Para comenzar descargamos el archivo
    • Y lo descomprimimos y nos queda una carpeta THM con todos los temas dentro
    • Copiamos la carpeta THM dentro de la raíz de nuestra memoria ej: ?:nombre de la carpeta donde le símbolo ? representa la letra de nuestra unidad
    • Y estaría listo
  2. Una vez listo vamos a nuestra PS2 abrimos el ulaunchelf
    • Para copiar nuestros temas (para el menú es y seleccionamos copy para copiar y Paste para pegar)
    • PS2 Fat
      1. Si lo copiamos al disco duro interno de la consola PS2
      2. Lo copiamos hdd0:/+OPL/ dentro encontraremos la carpeta THM y listo
    • memory card PS2
      1. al copiar la información a nuestra memory card solo podremos pasar entre 1 a 2 a nuestra memoria
      2. la ruta en nuestra memory card es mc0:/THM/ dentro copiamos el tema que nos guste
      3. y listo
    • Para nuestra USB
      1. Solo al copiar la información a la memoria estaría listo
  3. Una vez en terminado abrimos OPL
    • Presionamos start para entrar al menú
    • Y nos vamos a Display Settings y en Theme
    • Escogeremos nuestro tema que más nos guste
    • Le damos a OK y Save Changes
    • Y estaría listo
