PS2 Formatear Memory Card PS2

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32
  2. Consola PS2 con Chip Matrix, FMCB o Funtuna

Programas para este Tutorial

  1. FMCB
  2. FunTuna

Para Empezar este Tutorial

  1. Para esto si no tenemos podemos pasar el archivo de FMCB a nuestro USB
    • Una vez descargado nuestro archivo FMCB lo pasamos a la rais de nuestro USB ej: ?:nombre de la carpeta donde le símbolo ? representa la letra de nuestra unidad
  2. Conectamos nuestras memory card a nuestra consola en este orden
    • En memory card 1 nuestro FMCB o Funtuna
    • En memory card 2 y memoria a formatear
  3. Después nos vamos a uLaunchELF en nuestra consola
    1. Le damas al
    2. Y después nos vamos a la ruta: Ej MASS:/FMCB v1.966 Install (2019-04-13)/
    3. Y buscamos el archivo .elf ej: FMCBInstaller.elf
    4. Una vez dentro del programa
  4. Dentro del programa vamos a
    1. Presionamos y buscamos el mene donde dice Formatear T. M.
    2. Dentro del menú de formatear seleccionamos la Ranura 2 (la memoria a eliminar todo su contenido)
    3. Una vez seguros de la memoria a formatear le damos si y esperamos
  5. Y estaría lista para usar como nueva sin datos.
