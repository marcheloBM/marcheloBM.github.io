Para Empezar este Tutorial

  1. 1. Descargamos los archivos
    • a. Descomprimimos el archivo .rar o .zip
      1. i. PSP.NET.Server v1.0.zip
      2. ii. ps3netsrv_20220330.zip
    • b. Con la última versión disponible
    • c. Una vez descargado nuestro archivo
  2. En caso de no tener el FreeDVDBoot abriremos UltraISO u otro programa que nos permita quemar nuestra ISO al DVD (Grabar a la más baja velocidad permitida para logras un disco de calidad para nuestra consola).
    • Una vez listo lo colocamos en nuestra consola desbloqueada pero no cerramos la tapa.
  3. Una vez iniciado nuestro ulaunchelf
    1. Le damos al a FileBrowser
      • Si es el archivo .psu lo seleccionamos
      • Presionamos y le damas a Copy
      • Con para ir hacia atrás y vamos a MC0:/ con circulo
    1. Ahora seleccionamos la memory card para instalar los archivos o
    2. Seleccionamos
      • : instalación Manual (seleccionaríamos todas las configuraciones para después instalar todo)
      • : instalación automática (se instalará de forma automática)
    3. Presionamos para volver al buscador de la consola
