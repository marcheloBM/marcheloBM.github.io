PSP Firmware Oficial

Si queremos instalar el firmware oficial podemos seguir esta guía

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook (Para pasar Archivos)

Programas para este Tutorial

  1. .RAR Update Official
  2. .RAR Chronoswitch

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos los archivos .rar o .zip
      1. Descargamos el Update Official en referencia a nuestra consola
      2. PSP GO
      3. PSP X000 (PSP 1000, 2000, 3000)
      4. chronoswitch_7_0.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y copiamos
    • El Update Official solo lo copiamos a la raíz de nuestra memory stick y si queremos verificar el archivo esta es la ruta /PSP/GAME/UPDATE/
    • Y el archivo chronoswitch lo copiamos a la raíz de nuestra memory stick y si queremos verificar el archivo se encuentra en la ruta /PSP/GAME/chronoswitch_7_0
    • Una ves copiados los archivos a nuestra consola no vamos a nuestra consola
  3. Dentro de nuestra consola vamos
    • Juego/Memory Stick dentro encontraremos
    • Y buscamos la aplicación Chronoswitch Downgrader
    • Abrimos la aplicación una vez abierta le damos a la y de nuevo a la para continuar y de nuevo para iniciar
    • Después de que se inicie la consola nos mostrar la actualización y le damos a iniciar
    • Le damos a continuar con y aceptar y para terminar le damos a la
    • Se recomienda tener la batería cargada para poder realizar esta operación y conectar el cargador para no tener problemas
    • Una ves terminado le damos a la equis para terminar y esperamos que se reinicie la consola
    • Y estaría listo nuestra consola totalmente original
