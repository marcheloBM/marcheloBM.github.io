PSP Hack Permanente

Podemos liberar nuestra consola PSP para poder instalar aplicaciones o juegos y también la podemos dejar de forma permanente el hack.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook (Para pasar Archivos)

Programas para este Tutorial

  1. .rar Custom Firmware
  2. .rar INFINITY

Para Empezar este Tutorial

  1. Si contamos con un hack antiguo
    • Hay que desinstalarlo (Eliminar todo menos los juegos y aplicaciones)
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • Y si nuestra consola se encuentra en una versión antigua la actualizamos a la última versión disponible 6.61 nuestra guía les puede ayudar Firmware Oficial
  2. Descargamos los archivos
    • Descomprimimos los archivos .rar o .zip
      1. 6.61 PRO-C2.rar
      2. INFINITY 2.03 psp go o standard (el standard es para la versión x000)
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
    • Los copiamos a la nuestra memory stick
      1. 6.61 PRO-C2.rar contiene seplugins en la raiz y FastRecovery – PROUPDATE en /PSP/GAME por si queremos revisar
  3. Guía para habilitar el Hack y dejarlo Permanente
    • Con esta guía podemos tener nuestra consola PSP con el HACK permanente sin la necesidad de cuando uno la encienda volver a cargar el HACK nuevamente al inicio. Para esto seguiremos los siguientes pasos con los siguientes archivos en nuestra consola CFW PROUPDATE y INFINITY 2
      1. Ejecutar Aplicación PROUPDATE
        • Le damos a la después del proceso le damos de nuevo a la
      2. Ejecutamos INFINITY 2
        • Le damos a la como nos indica la aplicación y finalizamos con la para el Reinicio de la consola
      3. Volvemos a ejecutar PROUPDATE
        • Le damos a la después del proceso le damos de nuevo a la
      4. iv. Por última vez ejecutamos INFINITY 2
        • nos vamos hacia la y seleccionamos PRO CFW by Coldbird con la . Para terminar, le damos al botón HOME de la consola y a la para reiniciar
      5. Y estaría listo, desde el momento de iniciar la consola tendremos el hack activado para verificar solo de damos al botón y saldrá un menú de CFW de la consola.
  4. Y estaría lista nuestra consola para instalar juegos o aplicaciones
