Reparar de Windows

Método Express

Para reparar Windows de forma más resumida posible probar antes de ver las demás opciones

  1. CHKDSK: Revisar si hay sectores dañas en el disco para esto tenemos 2 formas de comprobar:
    • Desde CMD (Como Administrador)
      1. Comando: CHKDSK C:
      2. Revisará el sistema el sistema por completo y nos dirá si hay algún sector dañado del disco.
    • Desde el escritorio
      1. Clip derecho en el disco del sistema Propiedades
      2. En la ventana de Herramientas seleccionamos Comprobar y Examinar Unidad
      3. Después de esperar algunos minutos (dependiendo del sistema y disco) nos mostrara una venta
      4. Nos dirá si hay sectores dañados y si los hay los reparar
  2. Scannow: Repara Windows desde el cache de instalación de Windows si esta bien instalado repara todos los archivos dañados de windows
    • Desde CMD (Como Administrador)
      1. Ejecutamos el siguiente comando en CMD
      2. Comando: sfc /scannow
  3. Dism: La herramienta de administración y mantenimiento de imágenes de implementación (DISM) es un archivo ejecutable de línea de comandos a nivel de administrador disponible en Windows, el cual se puede utilizar para reparar la imagen de Windows o para modificar medios de instalación de Windows.
    • Desde CMD (Como Administrador)
      1. Ejecutamos el siguiente comando en CMD
      2. Comando: dism /online /cleanup-image /sacanhealth
      3. Pero si encuentra errores ejcutamos el siguiente comando
      4. Comando: dism /online /cleanup-image /RestoreHealth
  4. 4. Si las 3 fallan toca formatear el sistema operativo de Windows.

Método Para Windows

A través de CHKDSK

  1. Desde CMD (Como Administrador)
  2. Una vez abierta, escribiremos el comando tal y como hemos señalado en el apartado anterior. podremos utilizar varias opciones de forma consecutiva, para conseguir un completo análisis y reparación de nuestro disco duro.
  3. Comando: CHKDSK C: /f /r /x /v
  4. 4. En ocasiones no saldrá un mensaje que el proceso no podrá ejecutar pero se podrá hacer en el próximo reinicio del sistema le damos S
  5. Simultáneamente estaremos ejecutando varias de estas opciones para conseguir un mejor y más detallado resultado.

A través de SFC

  1. Si abrimos el menú inicio y escribimos “PowerShell” o “CMD” podremos acceder a las terminales en donde este comando puede ser ejecutado. En el resultado de búsqueda, tendremos que pulsar botón derecho sobre la opción para elegir “Ejecutar como Administrador”.
  2. Si escribimos en el terminal alguno de estos dos comandos, y pulsamos Enter:
  3. Comando: sfc
  4. Comando: sfc /?
  5. Obtendremos información acerca de este comando, así como toda la lista de opciones, que, situadas detrás del comando principal, nos darán determinadas funcionalidades.
  6. En la parte superior veremos su sintaxis básica, que será simplemente el comando principal, seguido de una barra y la opción, bien en minúscula o en mayúscula. SFC [/opción]
  7. Estas opciones serán las siguientes:
    • SFC /SCANNOW: utilidad principal para salvarnos el culo. Examina los archivos protegidos del sistema (los que están situados en C:\Windows\WinSxS) y los repara si es posible.
    • SFC /VERIFYONLY: con esta opción solamente verificamos la integridad de los archivos. Es útil para comprobar de forma más rápida que usando la opción anterior, si los archivos están dañados, en cuyo caso no lo notificará.
    • SFC /SCANFILE: verifica la integridad de un determinado archivo que nosotros le pongamos y lo repara, si es posible. Su sintaxis es “/SCANFILE=ruta archivo”.
    • SFC /VERIFYFILE: verifica la integridad del archivo, pero no emprende acciones de reparación. Su sintaxis es “SFC /VERIFYFILE=ruta archivo”.
    • SFC /OFFBOOTDIR: opción para reparar el arranque de Windows sin conexión. Sin conexión significa que accederemos a un disco duro que no esté en funcionamiento, por lo que esta opción tendremos que utilizarla desde un DVD de instalación de Windows, por ejemplo. Su sintaxis seria: “SFC /OFFBOOTDIR=ruta completa del archivo”.
    • SFC /OFFWINDIR: será lo mismo que lo anterior, pero para un directorio en concreto del disco duro.
    • SFC /OFFLOGFILE: reparación de un archivo de registro en las mismas condiciones que el caso anterior.
  8. En la mayoría de ocasiones utilizaremos simplemente SFC /SCANNOW, ya que es la opción que engloba todas las demás y que realiza un barrido completo y posterior reparación de los ficheros protegidos del sistema desde dentro del propio sistema (online) o con el disco en funcionamiento.

A través de DISM

DISM son las siglas de Deployment Imagen Servicing and Management y, tal como explica Microsoft, se trata de una herramienta de línea de comandos que se puede utilizar para reparar imágenes de Windows, lo que le permite solucionar los problemas derivados de una imagen de sistema corrupta sin tener que reinstalar el sistema operativo por completo.

  1. Escribe "cmd" (sin comillas) en el cuadro de búsqueda, Selecciona la aplicación Símbolo del sistema, haz clic en la opción Ejecutar como administrador
  2. Ahora, lo primero que tienes que hacer es comprobar el estado de salud de tu equipo. Para ello, introduce el siguiente:
  3. Comando: DISM /Online /Cleanup-Image /CheckHealth
  4. Espera unos segundos hasta que la comprobación se complete. Si aparece algún error antes de la finalización, es posible que tu PC esté dañado.
  5. A continuación, ejecuta la operación ScanHealth mediante este:
  6. Comando: DISM /Online /Cleanup-Image /ScanHealth
  7. Esto buscará buscará errores en el sistema. Espera hasta que la operación se complete.
  8. El siguiente paso consiste en ejecutar la operación RestoreHealth, que reemplazará todos los componentes dañados del sistema operativo e implementará una imagen nueva en tu PC. Esta operación llevará bastante tiempo, así que ten paciencia. Puedes iniciarla con este:
  9. Comando: DISM /Online /Cleanup-Image /RestoreHealth
  10. Después de utilizar DISM, tienes que usar la herramienta SFC (System File Checker), que verifica la integridad de los archivos y restaura por completo la salud de tu PC en función de la imagen implementada por DISM. Para ello, introduce el
  11. Comando: SFC /ScanNow
  12. y espera hasta que el proceso termine (tardará bastante, así que ten paciencia).
