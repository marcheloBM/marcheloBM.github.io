Empezando este Tutorial

  1. Debemos descomprimir y buscar el archivo
  2. Advanced_IP_Scanner_x.xx.xx y podemos ejecutarlo como administrador para mejor ejecución
  3. Podemos instalar el programa o podemos ejecutarlo sin instalarlo
  4. Una vez listo podemos Escáner de red fiable y gratuito para analizar LAN. El programa escanea todos los dispositivos de red, le da acceso a las carpetas compartidas y a los servidores FTP, le proporciona control remoto de las computadoras (mediante RDP y Radmin) e incluso puede apagar las computadoras de manera remota. Es fácil de usar y se ejecuta como una edición portable. Debe ser la primera opción para cada administrador de red.
