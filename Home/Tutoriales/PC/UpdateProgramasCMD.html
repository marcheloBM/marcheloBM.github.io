Empezando este Tutorial

  1. Para usar este método necesitamos buscar en el menú de inicio CMD y ejecutarlo si queremos lo podemos ejecutar como administrador
  2. Si queremos Actualizar los programas por CMD debemos comprobar si hay actualizaciones disponibles con el siguiente comando winget upgrade
  3. Si encuentra versión que no puede determinar usamos el siguiente comando winget upgrade --include-unknown
  4. Una vez tengamos la lista de programas a actualizar le damos al siguiente comando winget upgrade -–all
  5. Y le a esperar que complete la actualización y estaríamos listo, solo se pueden actualizar los programas de código libre dentro del sistema de winget.
