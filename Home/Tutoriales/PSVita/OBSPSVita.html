OBS en PSVITA

Plugin (p.o.c) de video UVC de PS Vita. Este es un complemento que te permite transmitir la pantalla de tu PSVita a tu computadora a través de USB.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Para descargar

Archivos Servidores
OBS Studio Programas Windows

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. OBS-Studio-??.?.?
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y copiamos
    • Tenemos que instalar OBS Studio en nuestra computadora
    • En nuestra consola
      1. Abrimos Auto Plugin 2 y buscamos el siguiente plugin
        • Udcd_uvc by xerpi
      2. Una vez instalado reiniciamos la consola para terminar la instalación
    • En nuestra PC
      1. Conectamos el USB a nuestra consola y al computador
        • Si el plugin quedo bien instalado la consola no aria nada
      2. Abrimos OBS Studio
        • Y podemos agregar todo de nuevo para el ejemplo
        • En Escenas le damos a agregar
        • En Fuente le damos a Agregar/Dispositivo de captura de video
        • Y le damos a Crear nuevo y le asignamos el nombre
        • En la ventana de configuración le damos a PSVita
        • Y le damos aceptar
        • Y listo lo demás es a comodidad de cada persona
