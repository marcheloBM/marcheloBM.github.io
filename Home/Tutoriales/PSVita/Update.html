Empezando este Tutorial

  1. Descargamos los archivos
    • Descomprimimos los archivos .rar o .zip
      • CMASetup
      • 3.74
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y copiamos
    • Para empezar en el computador instalamos CMASetup
      1. Comprueba que el asistente del gestor de contenido se esté ejecutando en la computadora
      2. Busca el ícono del asistente del gestor de contenido en el área de notificaciones de la barra de tareas.
      3. Conecta la PS Vita a la computadora mediante un cable USB
      4. La PS Vita comprobará automáticamente en Internet si tienes instalada la versión más reciente del software del sistema. En caso de haber un archivo de actualización de una versión posterior, esta se descargará en el sistema. Sigue las indicaciones para completar la actualización.
    • Por el archivo 3.74 (PSVUPDAT.PUP)
      1. En la unidad flash USB, crea las siguientes carpetas para guardar el archivo de actualización. En una computadora, crea una carpeta con el nombre “PSVITA” en la unidad flash USB. Dentro de esa carpeta, crea otra carpeta llamada “UPDATE”.
      2. Descarga el archivo de actualización y guárdalo en la carpeta “UPDATE” que creaste en el paso 1. Guarda el archivo con el nombre “PSVUPDAT.PUP”.
      3. Apaga completamente el dispositivo PS TV.
      4. Mantén pulsado el botón de encendido del dispositivo PS TV durante al menos siete segundos para encenderlo. El sistema se inicia en el modo seguro
      5. Conecta un control mediante un cable USB y, a continuación, presiona el botón PS.
      6. Selecciona Actualizar software del sistema > Actualizar desde dispositivo de almacenamiento USB.
      7. Conecta la unidad flash USB que tiene el archivo de actualización a tu dispositivo PS TV.
      8. Sigue las indicaciones para completar la actualización.
