Apollo Save Tool Vita

Apollo Save Tool es una aplicación para administrar archivos de partidas guardadas, trofeos y licencias en PlayStation Vita.

Empezando este Tutorial

  1. Descargamos los archivos
    • Descomprimimos los archivos .rar o .zip
      • apollo Save Tool v?.??.vpk
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y copiamos
    • Para pasarlo a nuestra consola lo podemos pasar mediante USB o FTP
    • Iniciamos VitaShell y le damos al dependiendo de lo hemos configurado nos saldrá USB o FTP.
    • Una vez pasada a nuestra consola lo instalamos el archivo .vpk en nuestra consola
    • Terminada la instalación salimos de VitaShell y en el menú de la consola tendremos el nuevo Icono flotando
  3. Buscamos la nueva Burbuja de nuestra aplicación
    • Abrimos Apollo Save Tool para poder usar sus funciones
    • Ahora tenemos algunas funciones para ir probando y testeando
      1. Trophies: Todos los trofeos de nuestros juegos
      2. Ext Saves: los SaveDatas que le pasemos a nuestra consola
      3. Int Saves: los SaveData de nuestra consola
      4. OnLine DB: una base de datos en línea para poder descargar SAVEDATA
      5. Tools: algunas herramientas para poder usar en nuestra consola
      6. Settings: algunas configuraciones para la aplicación
      7. About: información de los creadores de esta aplicación
    • Datos de las ubicaciones

      • Play Station Vita

      Play Station Vita Carpeta
      El almacenamiento externo guarda Tus partidas guardadas deben almacenarse en (uma0/imc0/xmc0/ux0):data/savegames/.
      El almacenamiento del usuario guarda Los juegos guardados se escanearán desde ux0:user/00/savedata/.

      PSP

      PSP Carpeta
      El almacenamiento externo guarda Tus partidas guardadas deben almacenarse en (uma0/imc0/xmc0/ux0):data/savegames/.
      El almacenamiento del usuario guarda Los juegos guardados se escanearán desde ux0:pspemu/PSP/SAVEDATA/.

      PS1

      PS1 Carpeta
      Guardados externos Tus partidas guardadas deben almacenarse en (uma0/imc0/xmc0/ux0):data/PS1/SAVEDATA/.
      Tarjetas de memoria virtuales Las imágenes VMC se escanearán desde (uma0/imc0/xmc0/ux0):data/PS1/VMC/.
    • Y listo a probar solamente
