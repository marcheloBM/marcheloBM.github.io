Liberar PSVITA

Liberar PSVita para poder instalar todos los programas y juegos gratis

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Para descargar

Archivos Servidores
Nano-Hencore Mega
QCMA Mega
VitaDeploy Mega
enso365 Mega
AutoPlugin2 Mega

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. Nano-Hencore.exe
      2. Qcma_setup-0.4.1.exe
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Antes de ejecutar lo primero es revisar que nuestra consola este en versión 3.74
    • De los programas instalamos QCMA en nuestro computador
    • Una vez instalado se nos mantiene en la barra inferior ejecutándose
    • En nuestra PSVita debemos tener una cuenta iniciada
    • Después instalamos Nano-Hencore en nuestro computador y se inicia
    • Ahora nos está pidiendo nuestro ID de PSVita
      1. Para saber cuál es nuestro ID nos vamos a Gestor de contenido en nuestra PSVita
      2. Nos vamos a Copiar Contenido y después le damos a PC
      3. Y en la siguiente pantalla podemos elegir por Cable USB o por WIFI
      4. Para este ejemplo lo voy a conectar por WIFI para después seleccionar mi PC
      5. Y nos pedirá colocar un código y en la pantalla de nuestro computador nos aparecerá un código
      6. Y ese código lo colamos en nuestra consola y se nos registra nuestro PC
      7. Para ver el código ID vamos a nuestros documentos/PS Vita/APP y veremos una carpeta con un código ese es nuestro código ID
    • Copias el ID y lo pegamos en el programa y le damos enter y nos abrirá el navegar con un código copiamos el siguiente código
    • Volvemos al programa y pegamos el nuevo código copiado
    • Y esperamos que nos prepare todo para nuestra consola de forma automática cuando termine nos mostrar una venta en nuestro computador
    • En la carpeta PCSG90096 la copiamos completa y la pegamos en documentos/PS Vita/APP y entramos a la carpeta con el código y la pegamos dentro
    • Cuando esté listo nos vamos al icono de QCMA y le damos clip derecho y le damos a Refrescar base de datos
    • Y con esto estaría todo listo ahora vamos a nuestra PSVita
    • En nuestra PSVita volvemos al Gestor de contenido/Copia contenido recomendó cerrar y abrir de nuevo a aplicación
    • Y le damos a Copiar contenido de PC al Sistema PSVita
    • Le damos a Aplicaciones/PS Vita y copiamos la aplicación h-encore
    • Una vez termina de copiarse buscamos la burbuja en la pantalla de inicio de nuestra consola
    • La iniciamos si nos sale una pantalla con letras estamos bien, pero si sale en banco hay que volver a iniciarla hasta q nos aparezca esta venta
    • En este menú le damos a
      1. Install HENkaku y esperamos
      2. Después le damos a Download VitaShell
      3. Y después le damos a Exit para salir
    • Puede ser que la consola se reinicie solo esperamos y después
    • Para esto abrimos h-encore y después le damos a Reset taiHen config.txt confirmamos y después le damos a Exit
    • Ahora para dejarlo de forma permanente vamos a Ajustes/Ajuste de HENkaku y verificamos que estén las primeras 3 casilla marcadas
    • Después abrimos Versión para Spoofing y lo demos en 3.74
    • Después en el menú principal abrimos VitaShell después le damos al botón y conectamos el cable (o Wifi si queremos configurarlo) de nuestra consola hacia el PC
    • Si no se nos ven todas las carpetas en nuestro ordenador le damos a Vista/Opciones y le damos a cambiar opciones de carpeta y búsqueda despues le damos a Ver y lo demos igual que la imagen
    • Después copiamos la aplicación VitaDeploy a nuestra consola
    • En nuestra consola cancelamos la transferencia y vamos donde dice ux0 y buscamos la aplicación y la instalamos y la abrimos
    • Podemos Instar algunas App downloader
      1. Enso installer
      2. Adrenaline
      3. PKGJ
      4. Y le damos a Download the selected apps cuando estemos listo
      5. Y esperamos que se descargue todas las aplicaciones y después las instalamos una a una
      6. Una vez instaladas todas salimos del todo
    • Abrimos VitaDeploy para bajar de versión
      1. Le damos a Install a different OS para bajarla de versión
      2. Le damos a Quick 3.65 install
      3. Ahora solo esperamos que termine, cuando termine nos da unos datos
      4. Ahora solo le damos a para confirmar la instalación y esperamos unos 20 segundos aprox y le damos de nuevo a la para aceptar
      5. De forma automática nos va a instalar la versión 3.65 y esperamos el reinicio
      6. Ahora nos falta hacerla permanente el desbloque para esto iniciamos h-encore y una vez inicia le damos a Reset taiHen config.txt y después le damos a Exit
      7. Después abrimos enso después le damos a y después le damos a
      8. Si nos da problema le podemos pasar el enso365 e instalarlo para eso hay que desinstalar en enso para instalar el nuevo
      9. Una vez instalado el nuevo enso la iniciamos y le damos a y después a la y después pulsamos un botón para reiniciar y estaría listo
      10. Y estaría listo cada vez que iniciamos muestra consola estaría desbloqueada
    • Iniciamos la aplicación adrenaline para que instale el emulador de PSP en nuestra consola
      1. La volvemos abrir para que termine de instalar el emulador
      2. Cuando termine presionamos para terminar
      3. Y listo
    • Ahora instalamos ITL-Enso le damos a Install the full iTLS package y esperamos que instale de forma automática instale todo y se reinicia la consola
    • Ahora vamos a instalar AutoPlugin2 ahora la conectamos para pasarle el archivo
    • Una vez instalado lo abrimos y le colocamos
      1. Le damos a Plugins de PS Vita/Instalar Plugins
      2. Y con el vamos a ir marcando lo que necesitemos
        • Adenaline by TheOfficialFloW (Fix double touxh)
        • Si queremos más revisa nuestra WEB AutoPlugins2
      3. Ahora le damos al último marcado para instalar todos automáticamente y lo reiniciamos para estar listo
    • Para descargar juegos vamos a PKGJ para descargar juegos
  3. Y listo para usar todos
