Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. Flycast
      2. kubridge (Plugin)
      3. Sega Dreamcast Bios (DC BIOS)
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y copiamos
    • Para pasarlo a nuestra consola lo podemos pasar mediante USB o FTP
    • Iniciamos VitaShell y le damos al dependiendo de lo hemos configurado nos saldrá USB o FTP
    • Una vez pasada a nuestra consola lo instalamos el flycast.vpk en nuestra consola
    • El Plugin Tendremos que instalar el Plugin kubridge de forma manual
      1. Lo pasamos a ux0:/tai el archivo kubridge.skprx
      2. Hay que reiniciar la consola para que funcione este plugin
    • Las BIOS Sega Dreamcast Bios (DC BIOS) se pasan a ux0:data/flycast/data/ los dos archivos
    • Las Roms podemos colocarlas en ux0:data/flycast/ para no tener problemas de búsqueda
    • Terminada la instalación salimos de VitaShell y en el menú de la consola tendremos el nuevo Icono flotando
  3. Buscamos la nueva Burbuja de nuestra aplicación
    • Abrimos Flycast ennuestra consola
    • Podemos empezar a cargar nuestros juegos, pero antes debemos configurar esto en nutra aplicación
    • Ahora solo seleccionamos el juego desde el menú principal
    • Y a disfrutar
