Empezando este Tutorial

  1. Descargamos los archivos
    • Descomprimimos los archivos .rar o .zip
      • Adrenaline v7.vpk
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y copiamos
    • Para pasarlo a nuestra consola lo podemos pasar mediante USB o FTP
    • Iniciamos VitaShell y le damos al dependiendo de lo hemos configurado nos saldrá USB o FTP.
    • Una vez pasada a nuestra consola lo instalamos el archivo .vpk en nuestra consola
    • Terminada la instalación salimos de VitaShell y en el menú de la consola tendremos el nuevo Icono flotando
  3. Buscamos la nueva Burbuja de nuestra aplicación
    • Abrimos Adrenaline para ver cómo funciona
    • Al iniciar la aplicación por primera vez veremos esto en nuestra pantalla
    • Nos estará solicitando que instalemos el firmware de la consola PSP para esto le damos a la para descargar
    • Cuando termine se nos cierra la aplicación la abrimos de nuevo para iniciar la instalación y le damos a la para instalar el firmware y comenzara la creación de directorios para la consola
    • Una vez terminada pulsamos para terminar y se iniciara la aplicación
    • Solo nos queda configurar el idioma, zona Horaria, fecha y hora y mas
    • Y listo podemos comenzar a usar la consola
  4. Igual revisamos si tenemos instalado el Plugin
    • Buscamos en AutoPlugin2 el siguiente plugin Adrenaline by TheOfficialFloW (Fix double touch) y lo instalamos
    • Para mejorar el funcionamiento de adrenaline
  5. Si Tenemos error al instalar el firmware podemos solucionarlo de esta manera
    • Para esto descargarnos el archivo 661.PBP después lo movemos a ux0:app/PSPEMUCFW
    • Volvemos a iniciar adrenaline y empezara la instalación
    • Y listo
