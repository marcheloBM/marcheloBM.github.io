Cambiar el HDD PS4

Si quieres cambiar el Disco Duro por uno de mayor capacidad

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores
PS4 Update Mega
WEB Oficial playstation.com
WEB NO Oficial darthsternie.net

Para Empezar este Tutorial

  1. Antes de cambiar el disco por uno nuevo o de mayor capacidad debemos
  2. Debemos hacer una copia de seguridad de nuestros datos para esto
    • Vamos a ir a nuestra página donde activaremos el BD Backup Nuestra WEB
    • Una vez echo nuestra copia en nuestro USB podremos restaurar los archivos a nuestro nuevo disco con el botón BD Restore
    • Esto es mas que nada al instalar el nuevo disco queremos restaurar los archivos
  3. Podemos crear una copia de seguridad de nuestra consola
    • Para esto vamos a Ajustes/Sistema/Copia de seguridad y restauración y vamos a Crear copia de seguridad de PS4
    • Una ves creada después restauramos a nuestro nuevo disco instalado
  4. Para prepara el cambio del disco
    • Debemos descargar el UPDATE de nuestra consola, pero el de reinstalación para esto vamos a PS4 Update
    • Una vez listo nuestro USB cambiaremos el disco de nuestra consola
    • Cuando iniciamos nuestra consola con el nuevo disco seguiremos los pasos del modo recovery de nuestra consola
    • Una vez terminada la instalación restauraremos la copia de seguridad echa con anterioridad
    • Una ves restaurada la copia volveremos a tener todo como estaba antes del cambio del disco
