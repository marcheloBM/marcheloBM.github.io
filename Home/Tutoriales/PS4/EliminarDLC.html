Eliminar DLC en PS4

Como Eliminar DLCS de tu PS4

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores

Para Empezar este Tutorial

  1. Para poder eliminar algún DLC debemos hacer lo siguiente pasos
  2. No vamos a Ajustes/Debug Settings/Game/Add Content Manager/Delete
  3. Al cargar la lista de juegos podemos seleccionar el que queramos eliminar los DLC por ej Resident Evil 4
  4. Podemos seleccionar todos o solo algunos y después le damos al botón Eliminar y estaría listo
  5. Para eliminar los temas hay que buscar una carpeta llamada Otros y dentro estarán los temas instalados en nuestra consola
  6. Y listo
