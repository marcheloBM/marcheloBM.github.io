PARAM.SFO Editor

PARAM SFO Editor podemos saber más información de nuestros juegos con param.sfo y podemos modificar más información.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores
PARAMSFO Editor Mega
WEB Oficial github.com

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. PARAM.SFO_Editor.exe
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Abrimos nuestra aplicación con PARAM.SFO_Editor.exe
    • Una vez abierta pasamos nuestro PARAM.SFO de nuestra Juego o aplicación (PS3, PSVita, PS4)
    • Una vez abierto nuestro archivo nos muestra información de nuestros juegos o aplicación que podemos modificar
    • Una vez modificado podemos guardar el archivo desde el SAVE que tiene
  3. Y esta riamos listo
