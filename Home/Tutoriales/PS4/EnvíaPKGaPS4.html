Envía archivos PKG

Envía archivos PKG a tu PS4 con Remote PKG Sender v2 podemos enviar cualquier archivo PKG desde nuestro pc sin la necesidad de una unidad USB con el siguiente programa.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores
PS4 Remote PKG Sender Mega
WEB Oficial github.com

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. PS4.Remote.PKG.Sender.V??????.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Para eso abrimos PS4 Remote PKG Sender V2.exe lo podemos abrir como administrador también y nos preguntara algo la primera vez que lo abramos
    • Cuando se habrá vamos a Config y le colocamos la IP de nuestra PC y le damos a PKG Base Path para buscar la ruta donde tenemos nuestros PKG y después le damos a Scan sub directories para escanear todos los directorios, en PS4 IP colocamos la IP de nuestra consola y subimos el Request Timeout para evitar errores y le damos a Apply para aplicar los cambios
    • Después vamos a Processing Center y vamos a check Server para ver como esta el servicio y si le damos a check PS4 y si nos muestra un error debemos abrir Remote PKG installer en nuestra consola
    • Ahora nos debería salir correcto el test en el programa, si esta todo bien vamos a Server donde veremos todos los PKG disponible para enviar solo le damos al Mas para ir enviado a nuestra consola
    • Después nos vamos a Processing Center y veremos los PKG que podemos enviar a nuestra consola solo le damos a Pley para empezar el envió
      • +
    • Y los juegos van a ir instalándose en nuestra consola PS4 automáticamente
