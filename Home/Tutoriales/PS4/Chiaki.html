Chiaki

PlayStation 4 de forma remota y sin abandonar la comodidad de los dispositivos Android.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores
Chiaki Mega
WEB Oficial git.sr.ht/

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. Chiaki-v?.?.?-Windows-x86_64.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. 2. Lo extraemos y ejecutamos
    • Para PC necesitamos
      1. Descargar la aplicación Chiaki-v?.?.?-Windows-x86_64.rar
      2. Solo abrimos chiaki.exe para ver sus funciones
      3. Si nos sale nuestra consola solo le damos doble clip para configurar el acceso
      4. Si no nos sale nada podemos buscarla manual mente
        • Solo le damos al botón MAS arriba a la derecha
        • Y después le colocamos la IP de nuestra consola es solo buscar en redes para saber cual IP tiene nuestra consola
      5. Si no abre la venta de Register Console solo nos que da
        • Identificar en que versión se encuentra nuestra consola
        • Dependiendo de la versión que tengamos nos pedirá
          1. PSN Online-ID el nombre del usuario de la consola
          2. PSN Account-ID (base64) El ID de PSN codificado con base 64
          3. Y el pin de nuestra consola
        • Para este ejemplo use PS4 >=8.0 y buscando por la WEB encontré en esta WEB PSN Account ID donde colocamos el nombre de usuario y no da el ID en Base64
        • Ahora solo colomas el PIN que se encuentra Ajustes/Ajuste de conexión del Uso a distancia/Añadir dispositivo<
        • Si todo nos sale bien nos mostrar el mensaje que se registro y podremos conectarnos a nuestra consola
        • Y listo podemos conectarnos a nuestra consola
    • Para Android necesitamos
      1. descargar la aplicación Chiaki-Up en nuestro Celular
      2. una ves descargado lo iniciamos para ver sus funciones
      3. cuando nuestra consola este encendida nos mostrar la información solo la seleccionamos y la cargamos para configurarlo
      4. al seleccionar nuestra consola solo debemos seleccionar la versión de nuestra consola por ej si estamos en 9.00 elegimos PS4> 8.0 y colocamos el nombre del Usuario en Base64 que lo obtenemos de la WEB PSN Account ID y le colocamos el PIN de Ajustes/Ajuste de conexión del Uso a distancia/Añadir dispositivo
      5. cuando todos los datos sean correctos nos muestra una pantalla de registro exitoso
      6. y nos conectara nuestro celular a nuestra consola PS4
      7. y Listo
