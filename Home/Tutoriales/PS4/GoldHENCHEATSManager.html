Empezando este Tutorial

  1. Descargamos el archivo lo descomprimimos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos al Menú de GoldHEN/Debug Settings/Package Installer
    • Seleccionamos el o los archivos PKG de nuestro USB y le damos a la
    • Y instalamos nuestra aplicación
    • Una vez instalada nos saldrá en el menú principal
  2. Una vez instalado
    • Abrimos GoldHEN CHEATS Manager para actualizar todo el contenido
    • Al abrir se actualizará la Base de datos de trucos a su última versión disponible
    • Si queremos actualizar manual solo le damos al botón Update para comprobar y descargar de nuevo los archivos
    • En Cheats vemos todos los trucos que tengamos en nuestra consola podemos revisar los trucos para ver su contenido
  3. 3. Con chronoss09
    • a. Podemos descargar los archivos GoldHEN CHEATS chronoss09 para obtener los cheats
    • b. Lo instalamos y lo abrimos
    • c. Y listo
  4. Para Activarlo
    • Debemos habilitar el exploit GoldHen o superior
    • Abrimos un juego en nuestra consola y debemos mantener el botón presionado por 3s aprox.
    • Nos saldría el menú del juego que tengamos abierto y los cheats disponible en la versión del Juego
    • Solo activamos las casillas de los trucos que queramos con y le damos al para volver al juego
    • Es solo ir probando cada truco
  5. Para Descargar la base de datos manual mente
    • Descargamos el archivo GoldCheats-Updater PC y lo abrimos desde nuestro PC
    • Le damos al botón Update all cheats para descargar todos los trucos disponibles
    • Cuando termine de descargar todos los trucos colocamos la ip de nuestra consola en IP Here
    • Y le damos al botón Upload all cheats to the PS4 para pasar toda la lista a nuestra consola
    • Solo nos quedaría activar en nuestro juego
  6. Si queremos pasarle base datos manualmente
    • Podemos descargar el archivo GoldHENCHEATSBD y pasarles los archivos
    • Para pasar los archivos, la mejor forma es por FTP para esto podemos usar FileZilla Client y la ruta a buscar
      1. Para los archivos SHN /user/data/GoldHEN/cheats/shn
      2. Para los archivos JSON /user/data/GoldHEN/cheats/json
      3. Para los archivos MC4 /user/data/GoldHEN/cheats/mc4
    • Y a buscar nuestro juego
  7. Para liberar mejor rendimiento con los Patches plugin
    • Para esto vamos a GoldHEN CHEATS Manager actualizamos el programa
    • Para instalación Manual HDD
      1. Descargar parche zip
      2. Copiar patch1.zipa /data/GoldHEN/ usando un cliente FTP.
      3. Abra GoldHEN Cheat Manager y haga clic en Actualizar.
      4. Si los parches se instalaron correctamente desde USB, debería ver el siguiente mensaje:
    • Instalación manual (sin conexión a través de USB)
      1. Descargar parche zip .
      2. Copiar patch1.zipa la raíz de la unidad USB.
      3. Abra GoldHEN Cheat Manager y haga clic en Actualizar.
      4. Si los parches se instalaron correctamente desde USB, debería ver el siguiente mensaje:
    • Para activar podemos usar GoldHEN CHEATS Manager o Itemzflow Game Manager
    • En GoldHEN CHEATS Manager
      1. Buscamos Patches y buscamos el juego para activar los parches
      2. Y activamos los trucos de nuestro juego
      3. Y abrimos el juego que activamos los parches
    • En Itemzflow Game Manager
      1. Abrimos el juego que queremos activar los parches
      2. Activamos los parches que necesitamos y abrimos el juego
  8. Trucos para Juegos de PS2
    • Siempre tener la última versión de GoldHen 2.2.4 o superior
    • Para esto descargamos GoldHENCheatsPS2 y lo instalamos
    • Lo iniciamos GoldHEN Cheats PS2 by Chronoss
    • Nos mostrara un menú donde podemos instalar (Install Cheats) o salir
    • Le damos a la opción Install Cheats y esperamos que instale todo el contenido
    • Y la consola se reinicia solo nos queda buscar nuestro juego para cargar los trucos
  9. Listo a disfrutar
