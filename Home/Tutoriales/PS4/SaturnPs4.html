Sega Saturn a Ps4

Podemos convertir juegos de sega saturn para nuestra PS4

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores
Saturn PS4 Mega
WEB Oficial logic-sunrise.com

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. Saturn2PS4_v?.?
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Una vez extraído veremos los siguientes archivos
    • Cuando tengamos nuestro juego lo pasmaos a la carpeta game
    • Si queremos colocar las imágenes de icono y de fondo solo las buscamos y las colocamos dentro de la carpeta game
    • Ahora le damos a Create_Saturn_PKG_v1.1.bat para ejecutar nuestra herramienta
    • Ahora le damos al idioma de preferencia
    • Después le damos a la opción 1 Crear un PKG de juego de Saturn
    • Presionamos una tecla para continuar
    • Le asignamos un ID para el PKG por ej 12345
    • Después le damos el nombre del juego
    • Después colocamos la portada del juego, para esto es más fácil arrastrar la imagen al CMD el fondo.jpg
    • Ahora colocamos el fondo
    • Después le damos a la opción 5 Compilar su Juego Saturn PKG
    • Y esperamos q termine el proceso
    • Después en la misma carpeta de la aplicación tendremos el archivo PKG
    • Solo nos queda instalarlo en nuestra consola lo pasamos a un USB en exFat
    • Y listo
