ICON MASK

Icon Mask es una sencilla aplicación Homebrew de PS4 que te permite modificar la apariencia de tus iconos aplicando una máscara PNG sencilla sobre ellos.

Para esta Guia

Requisitos para este tutorial

  1. USB en formato ExFAT

Programas para este Tutorial

Archivos Para descargar

Archivos Servidores
ICON MASK PKG Mega
WEB Oficial github.com

Empezando este Tutorial

  1. Descargamos el archivo lo descomprimimos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos al Menú de GoldHEN/Debug Settings/Package Installer
    • Seleccionamos el o los archivos PKG de nuestro USB y le damos a la
    • Y instalamos nuestra aplicación
    • Una vez instalada nos saldrá en el menú principal
  2. Una vez instalado
    • Abrimos Icon Mask para configurarlo
    • Podemos colocar más en una carpeta ICON en nuestro USB y dentro podemos colocar mas
    • Para ir cambiando de estilos vamos cambiando con o para ir buscando el que más nos guste
    • Cuando encontramos le damos en aplicar
    • Si queremos cambiar un icono de los que descargamos tiene que estar 512x512
    • Para cambiar uno le damos a Pre-made Icons
      1. En la parte de arriba con o buscaríamos el juego para cambiar el icono
      2. En la parte de abajo con o seleccionamos el icono que queremos cambiar y le damos a aplicar
    • Y listo
