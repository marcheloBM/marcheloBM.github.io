Empezando este Tutorial

  1. Versiones compatibles
    • Debemos tener nuestra consola en una versión firewall especifica 5.05, 6.72, 7.00, 7.02, 7.50, 7.51, 7.55, 9.00 y 11.00 hasta la fecha pueden salir más, pero hay que darle tiempo
    • Si tenemos nuestra consola en una versión distinta como por ejemplo en versión 8.50 solo nos quedaría subirla de versión, para esto solo nos queda actualizarla de forma manual Update PS4
  2. Preparando la consola
    • Para poder seguir debemos activar el internet de nuestra consola, pero al hacer esto nuestra consola puede actualizarse de forma automáticamente
    • Para esto conectamos el internet y cancelamos lo más rápido posible la descarga de la actualización
    • Abrimos el navegador y abrimos cualquier host disponible de PS4
      1. MarcheBMPC marchelobm
      2. Gamerhack gamerhack
    • Buscamos el Payloads y dentro buscamos el Disable Updates
    • Con esto deshabilitamos las actualizaciones del sistema cada ves que encendamos la consola
  3. Para habilitar el Exploit
    • Solo vamos al navegador de nuestra consola y abrimos los siguientes links
      1. https://marchelobm.github.io/ desde 5.05 a 9.00
      2. https://nanospeedgameryt.github.io/ solo 9.00
      3. http://thewizwikii.github.io/PS4/9.00/ solo 9.00
      4. https://gamerhack.github.io/index.html desde 5.05 a 9.00
    • Solo buscamos la versión de nuestra consola y el ultimo GoldHen disponible y a espera que se desbloque nuestra consola
  4. Y listo
