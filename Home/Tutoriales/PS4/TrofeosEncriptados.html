Trofeos encriptados para PKG

Verificar los Trofeos correctamente si decides construir pkg por error en los Trofeos en nuestros juegos de PS4.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores
Trofeos encriptados para PKG Mega

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. PS4-Fake-PKG-Tools-3.87-main.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Al descomprimir dentro de la carpeta tendremos todas las aplicaciones disponibles
    • Ahora buscamos nuestro juego dumpiado para verificar si este encriptado los trofeos
    • Para esto buscamos dentro de la carpeta la aplicación helper.exe para soltar la carpeta CUAS??????-app de nuestro juego encima de la aplicación
    • Automáticamente se nos abrirá una venta de CMD con la información recopilada, nos dirá si este encriptado o no nuestros trofeos con la imagen siguiente nos dices que esta todo correcto y no es necesario hacer nada mas
    • Y también aremos lo mismo con las actualizaciones la carpeta CUAS??????-patch y en esta ocasión nos dirá que están encriptado
    • Y nos dará la id de la carpeta de juegos lo cual lo anotamos
    • Ahora vamos a nuestra consola y habilitamos el FTP de nuestra consola PS4
    • Abrimos FileZilla y entraremos a nuestra consola y buscamos la siguiente ruta /user/trophy/conf y el id de nuestro juego y dentro copiamos el TROPHT.TRP a la carpeta donde este encriptado nuestro juego EJ CUAS??????-patch/sce_sys/trophy
    • Ahora le colocamos el nombre del antiguo y el antiguo lo eliminamos y nos quedaría el ultimo descargado por FTP
    • Ahora si repetimos los pasos d nos dirá que los trofeos están desencriptados
    • Si están todos listo la carpeta app y patch desencriptado estaría todo listo
    • Solo nos quedaría crea el PKG
  3. Y listo
