DNS + Cancelar Update

Para poder impedir la actualización de la consola o poder usar aplicaciones en PS4 debemos bloquear el acceso.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores

Para Empezar este Tutorial

  1. Por Debug Setting
    • Para ello desbloqueamos nuestra consola y nos dirigimos a Ajustes/Debug Settings
    • Y en Debug Setting nos dirigimos a PlayStation Network
    • Después en NP Environment
    • Y dentro cambiamos de “np” a “sp-int”
    • Y reiniciamos nuestra consola
    • Y listo (con esta configuración nuestra consola no se actualizará la versión y los juegos)
  2. Por DNS
    • Para esto configuramos una conexión a internet por medio de Ethernet o Wifi y colocamos las siguientes configuraciones
    • Podemos usar las DNS de AL-Azif que sería:
      1. DNS de Al-Azif 1.76 > 9.00
      2. DNS Primaria: 165.227.83.145
      3. DNS Secundaria: 192.241.221.79
  3. Por configuraciones
    • Vamos a Ajustes/Sistema/Descargas automáticas y revisamos si está marcado Archivos de actualización del software del sistema si esta lo desmarcamos
    • Lo otro es desmarcar Informar automáticamente de los errores del software del sistema para evitar comunicación con Sony
    • Ahora iremos a nuestra página de confianza para ejecutar nuestro Jailbreak WEB marchelobm.github.io/PS4 y usaremos el Disable Update para evitar las conexiones a Sony y que se nos actualice la consola
