Crear Avatar

En este tutorial les muestro como cambiar el Avatar o imagen de perfil sin PSN de forma Offline

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores
Auto Mega
Manual Mega

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. Auto
        • directx_Jun2010_redistAuto.exe
        • PS4 Avatar Maker – Lapy
      2. Manual
        • archivos para PS4.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
  3. En Auto
    • instalamos las aplicaciones directx_Jun2010_redistAuto.exe y lo ejecutamos
    • Lo instalamos y esperamos que termine
    • Después descomprimimos PS4 Avatar Maker - Lapy y ejecutamos PS4-Xplorer Avatar Maker.exe como administrador
    • Seleccionamos local (para cuentas no activadas) o Offine Activated (para cuantas activadas) solo ir probando
    • En la imagen central le damos doble clip para buscar la imagen (Imágenes cuadradas por ej 512x512)
    • Y para finalizar le damos al botón Export Avatar y lo guardamos en nuestro equipo y crearemos el .xavatar
    • Después lo pasamos al USB y lo instalamos en nuestra consola
    • Abrimos PS4-Xplorer para instalarlo en nuestra consola y lo buscamos el USB mnt/usb0
    • Seleccionamos la cuenta donde queremos instalar el avatar con dos
    • Y después le damos a salir de PS4 y buscamos el usuario que le instalamos el usuario
  4. El Manual
    • Descargamos las aplantillas archivos para PS4.rar
    • En la planilla tenemos las imágenes como referencia para editar
      • Imagen Tamaño
      Avatar.png Resolución: 440x400 pixeles
      avatar64.dds Resolución: 64x64 pixeles
      avatar128.dds Resolución: 128x128 pixeles
      avatar260.dds Resolución: 260x260 pixeles
      avatar440.dds Resolución: 440x440 pixeles
      online.json
    • podemos usar cualquier editor de imágenes que tengamos mientras sea compatible con .dds como sugerencia photopea
    • Dentro del archivo online.json copiaremos lo siguiente: {"onlineId":"XXXXXXX","firstName":"XXXXXXX"} donde Las X las sustituirán por su ID y su nombre de perfil y quedaría algo así {"onlineId":"Usuario","firstName":" Usuario "}
    • cuando tengamos listo los archivos los pasamos por FTP en la siguiente ruta: /system_data/priv/cache/profile
    • y le damos a salir de PS4 para ver el usuario que le pasamos las imágenes
    • y listo
