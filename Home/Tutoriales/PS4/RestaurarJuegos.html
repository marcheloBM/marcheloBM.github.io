Recuperar Juegos de PS4

Podemos recuperar juegos que se hayan eliminado de la vista, pero sigan en la consola

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores
PS4_db_Rebuilder_EXT Mega
WEB Oficial github.com

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. PS4_db_Rebuilder_EXT-0.1.zip
      2. python-3.10.5-amd64.exe
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Por copia de seguridad en USB
      1. Si no hemos perdido nada podemos hacer una copia de seguridad en nuestro USB
      2. Para esto vamos a nuestra página favorita Nuestra WEB donde ejecutamos nuestros Payloads y buscamos DB Backup
      3. Y si tenemos una copia echa con anterioridad la podemos recuperar con el botón DB Restore
      4. El problema de este método si instalamos mas juegos después de la copia estos juegos no se recuperarán
    • Por Aplicación
      1. Tenemos que habilitar el FTP en nuestra consola
      2. Instalamos el python-3.10.5-amd64.exe en nuestro equipo
      3. En nuestro PC descomprimimos PS4_db_Rebuilder_EXT-0.1.zip y en la ruta escribimos powershell
      4. Dentro de la ventana escribimos py fix_db.py 192.168.2.218 con la ip de nuestra consola
      5. Y se restauraría todos los juegos y aplicaciones
      6. Ahora en nuestra consola abrimos el navegador y lo cerramos y listo aparecerán todos los juegos perdidos
    • Por FTP
      1. Solo abrimos FileZilla y nos conectamos a nuestra consola
      2. Buscamos en la siguiente ruta /system_data/priv/mms y encontraremos el archivo app.db y lo copiamos a nuestro equipo
      3. Y si tenemos alguna copia echa con anterioridad la pasamos a la misma ruta
      4. Y listo
