Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores

Para Empezar este Tutorial

  1. Solo tenemos que tener activado GoldHEN en nuestra consola
  2. Para esto podemos abrir nuestra web preferida
  3. Y buscaremos WEB Activator esto podría tardar un rato en cargar
  4. Al usuario que queremos cambiar le damos clip al nombre y cambiara el código primero
  5. Después le damos al botón Set & Activate y esperamos el mensaje
  6. Ahora solo nos queda reiniciar la consola y podemos usar las funciones de una cuenta activada por ejemplo poder copiar partidas en un USB
  7. Y listo
