Cambiar los Iconos PS4

Podemos cambiar todos los iconos de nuestra consola para darle un nuevo estilo

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Servidores
Iconit Mega
WEB Oficial github.com

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. Iconit.v?.??
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Podemos descargar la platilla Circulo.png para ayuda de la forma como referencia y tamaño
    • Cuando tengamos listo nuestro diseño, podemos usar Adobe PhotoshopUltraISO
    • Lo guardamos como .PNG donde queramos
    • En nuestra consola activamos el FTP para ver la IP de nuestra PS4
    • En nuestro pc descomprimimos el archivo Iconit.v?.?? en su última versión disponible y ejecutamos el archivo Iconit v5.13.0.2 por ejemplo
    • Podemos cambiar el idioma del programa en Settings/Options… y en Default Language colocamos el idioma de preferencia y lo otro le damos a encendido le damos al botón Save y cerramos la aplicación y la abrimos de nuevo
    • Ahora nos conectamos a nuestra consola con la IP y el puerto y le damos a Connect PS4 y esperamos que habrá la ventana de iconos
    • En cambiar icono podemos seleccionar el icono recién creado y seguimos los pasos
    • Si llega a fallar puede ser que no este en formato .png el icono
    • Para revisar si quedo bueno le damos a cambiar usuario o en modo reposo y volvemos abrir el usuario y revisamos como quedo el Icono
