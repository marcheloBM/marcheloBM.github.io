PS3 Movian

Es una app de ps3 que se instala en CFW/HEN para ver peliculas, series, listas de reproducción y escuchar música, todo Online. Es tan útil que si combinas Movian con las apps de Spotify, Netflix y Crunchyroll te queda todo un centro multimedia completo.

Para Empezar este Tutorial

  1. Descargamos los archivos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos a Administrar archivos PKG
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • Y instalamos los 2 pkg de nuestra aplicación
    • movianxxxxxx.pkg y ps3Flix_code_veronica_x.pkg
    • Una vez instalada nos saldrá en el menú de Servicios de TV/Video
  2. Abrimos nuestra aplicación
    • Una ves dentro nos vamos a PS3 HDD
    • Buscamos la carpeta Ps3Flix y dentro encontraremos un plugin llamado PS3Ultimate.com y le damos para instalar el plugin
    • Nos vamos para atrás y vemos instalado el plugin lo abrimos con
    • Y encontraremos todas la películas y serie dentro de este contenido
  3. Vamos a USB Driver 0
    • Y dentro hay un archivo llamado spain1.m3u
    • Donde hay muchas tv en vivo para probar
    • Y también está la lista33.m3u
    • Donde hay mas tv en vivo
  4. Y listo a disfrutar y a descargar mas lista en formato .m3u para más listas actualizadas
