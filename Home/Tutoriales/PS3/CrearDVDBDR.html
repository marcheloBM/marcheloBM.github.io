Crear DVD o BD-R Juegos

Podemos crear copias de nuestros juegos en DVD o BD-R para nuestra consola con HEN

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip
  3. USB en FAT32 (Capacidad mayor al peso del Disco a copiar)

Programas para este Tutorial

Archivos Para descargar

Archivos Servidores
PS3 ISO TOOLS Mega
Cobra ODE GenPS3iso Mega
multiMAN Mega

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. PS3 ISO TOOLS V2.2.rar
      2. Cobra_ODE_GenPS3iso_v2.3.zip
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Para empezar
    • Copiar disco
      1. Abrimos multiMAN multiMAN para poder hacer una copia de nuestro juego favorito original
      2. Buscamos nuestro juego del disco y le damos a y después donde dice copiar
      3. Y le damos a copiar al HDD interno le damos a NO y después nos pregunta si queremos copiar al USB001 externo le damos a SI
      4. Una vez terminado le damos a NO a lo que nos pregunte
    • Dentro del USB
      1. Encontraremos una carpeta llamada GAMES y dentro el juego Dumper
      2. Lo copiamos a nuestro pc para trabajarlo de mejor manera
      3. Una vez copiado abrimos el programa Cobra_ODE_GenPS3iso_v2.3.zip
      4. Seleccionamos el juego recién dumper
      5. Seleccionamos donde queremos guardar nuestro juego
      6. El IRD le damos a cancelar y al Split le damos a cancelar
      7. Esperamos el proceso que terminara en una ISO del Juego
      8. Ahora solo nos quedara grabar nuestro eso ISO a un Disco
      9. Para esto solo usamos nuestro grabador preferido (Nero, ImgBurn)
      10. Le damos la menor velocidad permitida
    • En la consola
      1. Colocamos nuestro disco recién grabado
      2. Si nos manda error le damos a y le damos a buscar actualizaciones
      3. Y le damos a instalar a la nueva actualización
    • Y listo
