PS3 Make Package Custom

Podemos crear archivos pkg de lo que queramos como por ejemplo temas, juegos, licencias, etc

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

  1. RAR Make Package Custom

    Empezando este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. make_package_custom_1.2.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos
    • Antes de empezar hay una carpeta llamada custom si no está la creamos
    • La abrimos y dentro especificamos la ruta de lo que vamos a pasar (Tema, Música, juegos, licencia)
    • Como por ejemplo un Tema para esto necesitamos la ruta de instalación del contenido
    • Y crearemos las carpetas con la ruta: dev_hdd0/theme para esto son dos carpetas
      1. Carpeta 1: dev_hdd0
      2. Carpeta 2: theme(dentro de la carpeta 1)
    • Y en la carpeta theme pasamos el archivo de tema para nuestra consola
    • Una vez terminada de pasar todo el contenido crearemos el pkg
    • Para esto abrimos make.bat y esperamos que termine
    • Después pasamos el archivo recién creado a muestra USB y a la consola
  3. Solo nos quedaría instalarlo y probarlo
