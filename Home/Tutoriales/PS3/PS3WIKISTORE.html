PS3 WIKISTORE

Store donde podrás descargar juegos de PSX, PS2, PSP, PS3 PSN y Exclusivos directamente desde tu consola PS3. Tendrás acceso a descargas de Hombrew, emuladores retro y un montón de modificaciones para tu consola.

Empezando este Tutorial

  1. Descargamos los archivos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos a Administrar archivos PKG
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • Y instalamos estos pkg de nuestra aplicación
      1. VIDEOSTORE 2.?
      2. FIX HEN 4.?
    • PKG con la última versión disponible
    • Una vez instalada nos saldrá en el menú de Juegos
  2. Una vez instalada nuestra aplicación
    • Reiniciamos la consola o la apagamos y la encendemos de nuevo
    • Con este podemos ver la aplicación la abrimos
    • Para el uso de la aplicación es necesario inicial el HEN
    • Podemos descargar juegos de PS1, PS2, PS3, PSP, juegos retro y aplicaciones
    • Y a disfrutar la aplicación
