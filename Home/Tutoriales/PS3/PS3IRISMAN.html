PS3 IRISMAN

Gestor de juegos, administrador de archivos y muchas herramientas más componen este manager que lleva muchos años dando soporte en Playstation 3

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32
  2. USB en Formato NTFS

Programas para este Tutorial

  1. PKG IRISMAN
  2. WEB Oficial aldostools-IRISMAN

Empezando este Tutorial

  1. Descargamos el archivo lo descomprimimos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos a Administrar archivos PKG
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • Y instalamos nuestra aplicación
    • Una vez instalada nos saldrá en el menú de Juegos
  2. Una vez instalado la primera abertura de la aplicación
    • Podemos configurar el tamaño de la pantalla
  3. Con el botón podemos entrar a las configuraciones y herramientas
    • Podemos acceder a las siguientes configuraciones
    • Manejador de archivos
    • Instalar .PKG
    • Borrado de Ficheros Cache
    • Copia desde /dev_usb/iris a la carpeta de IRESMAN
    • Cobra / payload Sin Disco
    • Español
    • Spoof Console ID
  4. sí presionamos podremos acceder algunas herramientas de configuración para nuestro juego
    • las siguientes opciones son para los juegos
    • Config. Juego
    • Copiar Juego
    • Borrar Juego
    • Arreglar Permisos
    • Comprobar Ficheros
    • Crear ISO
    • Copiar a Favoritos
    • Actualización de Juegos
  5. IRISMAN incluye una función para pasar archivos PKG mayor de 4GB a nuestra consola
    • Para esto pasamos nuestro archivo PKG en una memoria USB en formato NTFS para poder copiar a la unidad
    • Una ves listo nuestra USB lo conectamos a nuestra PS3 y abrimos IRISMAN
    • Una ves en IRISMAN le damos a
    • Y vamos a Herramientas y a Manejador de Archivos
    • Una ves dentro tendremos nuestra consola en el lado A y en el lado B nuestro USB
    • En el lado A buscamos la carpeta dentro de la consola para dejar nuestro Archivos PKG y la ruta seria /dev_hdd0/packages
    • En el lado B buscamos el archivo PKG que vamos a copiar en la ruta /ntfs0:
    • Una ves listo le damos a para ver un menú donde le damos a Copy
    • No preguntara el proceso a continuar y le damos Si
    • Una ves copiado en nuestra consola podemos salir de IRISMAN
    • Y instalar nuestro archivo PKG
      1. Para instalar nuestro archivo PKG
      2. Vamos a Administrar archivo PKG / Instalar archivo PKG y en Almacenamiento del sistema PS3
      3. Se encuentra nuestros archivos recién copiados y podremos instalarlos
  6. Y estaríamos listo
