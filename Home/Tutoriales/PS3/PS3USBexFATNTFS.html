USB exFAT - NTFS

Con esta app podemos usar nuestros USB o HDD con el formato exfat o NTFS para nuestros juegos en formato ISO

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32

Programas para este Tutorial

  1. PKG prepISO
  2. WEB Oficial github.com

Empezando este Tutorial

  1. Descargamos los archivos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos a Administrar archivos PKG
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • Y instalamos estos pkg de nuestra aplicación
      1. prepISO_1.30.pkg
    • PKG con la última versión disponible
    • Una vez instalada nos saldrá en el menú de Juegos
  2. Para comenzar
    • Buscamos nuestra aplicación recién instalada preISO
    • Podemos dejar presionado la para dejar esta aplicación en la memoria cache de la consola y así si la apagamos no se perderá la configuración
  3. Para nuestro WEBMAN
    • También tenemos la aplicación instalada en Juegos de webMAN/configuración de webMAN para activarla
    • Podemos contarlo a nuestra consola PS3 nuestro USB o HDD en formato exFAT – NTFS
    • Nuestros juegos ISO tienen que estar ubicado en la carpeta PS3ISO
    • Por si no detenta nuestros juegos le damos a refrescar a webMAN y listo nos mostrara los juegos por ejemplo si es un disco en NTFS nos mostraría ntfs0/PS3ISO
    • Y estaríamos listo
