PS3 Plex

Una aplicación con la que puedes convertir tu ordenador en un centro multimedia utilizando el contenido digital que tengas en él. Reconoce todos los archivos multimedia que tengas alojados en la carpeta de origen.

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32

Programas para este Tutorial

  1. PKG Plex
  2. Oficial desde PlayStation Store en nuestra consola

Para Empezar este Tutorial

  1. Podemos descargar desde PlayStation Store y descargar Plex original es gratis
  2. Descargamos los archivos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos a Administrar archivos PKG
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • Y instalamos el pkg de nuestra aplicación
    • Plex-v1.01… con la última versión disponible
    • Una vez instalada nos saldrá en el menú de Servicios de TV/Video
  3. Configuramos nuestro equipo con el Plex
    • Podemos instalar nuestra aplicación en nuestro PC o Notebook o dispositivo compatible con plex
    • Una vez instalado y configurado podemos pasar nuestras películas y series para tenerlas disponible para nuestros dispositivos
  4. 4. Abrimos nuestra aplicación
    • a. Desde la consola y registramos el código en nuestra cuenta para activar
    • b. Una vez iniciada la sección podemos revisar nuestros archivos descargados en nuestros dispositivos (servidor o PC)
    • c. Y reproducir los archivos en nuestra consola en la misma red
  5. 5. Y a disfrutar nuestra aplicación
