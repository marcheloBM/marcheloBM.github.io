PS3 Coldboot

Crear Coldboot (Logotipo de Inicio) personalizados con las imágenes que más te gusten.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

Archivos Para descargar

Archivos Servidores
Coldboot Mega github.com
Make Package Custom Mega

Empezando este Tutorial

  1. Descomprimimos el archivo .rar o .zip
    • Descomprimimos el archivo .rar o .zip
      1. PS3 Coldboot.zip
      2. PS3 Coldboot.pkg
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    1. Make PKG Custom: programa para crear pkg para instalar
    2. COLDBOOT.png: imagen de referencia para editar
    3. COLDBOOT.psd: proyecto echo en Photoshop para edición
    4. Easy Static RAF Converter v1.0a.exe:
  3. Una vez tengamos todos los programas buscamos una imagen de alta resolución para no tener problemas en la edición de preferencia de 1920x1080 y lo guardamos en formato png
  4. Una vez listo abrimos el programa Easy Static RAF Converter.exe si nos pide descargar algo como esto le damos a descargar para mejor funcionamiento.
  5. una vez abierto nos pedirá que seleccionemos la imagen que dejamos lista anteriormente y nos mostrar una interfaz como esta
  6. Le podemos dar al botón Show Preview para visualizar como se verá en la consola
  7. Si terminamos de ajustar la imagen a nuestro gusto le damos al botón Convert y esperamos el mensaje de done en pantalla
  8. En donde está el programa nos creara otra carpeta con el nombre COLDBOOT y dentro los archivos
    1. Coldboot.raf: el archivo para nuestra consola
    2. Preview.png: vista previa
  9. Copiamos el archivo Coldboot.raf a nuestro programa Make PKG Custom dentro de la carpeta
    1. dev_blind: para consola con CFW
    2. dev_rewrite: para consola con HEN
  10. Movemos la carpeta que nos corresponde a custom para después copiar el archivo Coldboot.raf en la última carpeta que nos aparezca
  11. Nos debería quedar el archivo en la siguiente ruta custom\dev_rewrite\vsh\resource con el archivo Coldboot.raf
  12. Una vez el archivo dentro de la ruta volvemos a la carpeta del programa Make PKG Custom y ejecutamos make.bat y esperamos a que termine
  13. Cuando termine nos crear el archivo custom.pkg donde esta el programa solo nos queda instalar en nuestra consola y reiniciar para probar como nos quedo el cambio
  14. Por si queremos volver al original podemos descargar el archivo Coldboot original.pkg que se encuentra en la misma carpeta de descargar y instalar y reiniciar
  15. Y listo
