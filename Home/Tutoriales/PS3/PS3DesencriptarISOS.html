Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. PS3 Desencriptar ISOS.zip
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Tendremos que tener estos archivos después de extraerlos
    • También tendremos que tener la iso del juego de PS3 y la key como para este ejemplo
    • Los pasamos a la carpeta del programa para después ejecutarlo por cmd o powerShell
      1. Para powerShell: presionamos la tecla shift más clip derecho y le damos a Abrir la ventana de PowerShell aquí
      2. Para cmd: en la ruta al principio escribimos cmd como ejemplo
    • Dentro de la ventana que abrimos escribimos los siguiente
      1. El nombre del programa PS3Dec.exe seguido de un espacio
      2. Después colocamos d key y escribimos la key dentro del archivo que descargamos terminado en .dkey
      3. Y nos quedaría algo como esto, por ejemplo:
        • Cmd: PS3Dec.exe d key 7872FA59EEDB2FA9BE22132AC3996087
        • PowerShell: .\PS3Dec.exe d key 7872FA59EEDB2FA9BE22132AC3996087
      4. y después se arrastra la iso del juego dentro de la ventana (cmd o PowerShell), el código depende el juego no todos son iguales
      5. una vez terminado
      6. Nos crear un nuevo archivo con el término .iso.dec solo eliminamos el .dec del nombre para que nos quede solo el nombre.iso por ejemplo uDraw Studio - Instant Artist (USA) (En,Fr,De,Es,It,Nl).iso
      7. Lo pasamos al USB en la carpeta PS3ISO y a probarlo en nuestra consola PS3
      8. Y listo
