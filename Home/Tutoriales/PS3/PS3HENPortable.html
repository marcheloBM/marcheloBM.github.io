PS3 HEN Portable

Podemos instalar nuestro HEN desde sus celulares Android sin la necesidad de un PC o de conexión directa a WEB Oficial

Requisitos para el Tutorial

Requisitos para este tutorial

  1. Android
  2. Para descomprimir archivos .rar o .zip .rar o .zip

Programas para este Tutorial

  1. RAR HEN Portable

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. Podemos descargar una de estas opciones
        • Android-PS3-HEN-4.90.zip
        • PS3-Android-4.90.zip
      2. También descargamos
        • kWS.Android.Web.Server_v1.7.6_apkpure.com.apk
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Descomprimimos unos de los archivos .zip que hemos descargado por recomendación PS3-Android-4.90.zip (recordar la ubicación de estación)
    • Y instalamos kWS.Android.Web.Server_v1.7.6_apkpure.com o si tenemos otro lo podemos usar nuestro servidor WEB para nuestro Android
  3. Una vez que tengamos todo instalado
    • Abrimos kms y lo configuramos desde Setting
    • Nos vamos a los tres puntos y seleccionamos Home directory
    • Seleccionamos la carpeta donde extraemos los archivos si es en la memoria interna seria
    • Yo la carpeta final le cambie el nombre por PS3 (si queremos le cambiamos el Port por defecto es :8080)
    • Una ves listo le damos al botón Start server y nos mostrara el servidor arrancando
    • Como podemos observar nos mostrara un Host donde esa dirección la colocamos en el navegador de nuestra consola PS3
    • Seleccionamos el auto HEN
  4. Estaríamos listo solo esperar el reinicio de nuestra consola para disfrutar el HEN
