PS3netsrv Android

Con esta APP para Android nos permite transmitir las imágenes ISO desde nuestro celular hacia la consola.

Para esta Guia

Requisitos para este tutorial

  1. Android 4.0.1 o Superior

Programas para este Tutorial

Archivos Para descargar

Archivos Servidores
PS3netsrv Android Mega
WEB Oficial github.com

Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. ps3netsrv-android vxxxxx.zip
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos
    • Para este pasamos los siguiente a nuestro celular
      1. app-debug.apk
      2. Juegos PS3 en ISO
    • Los Juegos lo pasamos a una carpeta llamada GAMES por si las moscas
    • Instalamos la aplicación app-debug.apk en nuestro celular
    • Una vez instalado lo abrimos para configurarlo
    • Para configurar el programa nos vamos a los 3 puntos de la parte superior
    • En este menú podemos configurar la ruta de las ISO
    • Escogemos donde está ubicado los juegos por ej lo deje en la carpeta descargar
    • Y el juego se encuentra dentro de la carpeta GAMES
    • Una vez listo todas las configuraciones volvemos atrás para iniciar el Servidor
    • Para terminar, debemos configurar el webMAN de nuestra consola
      1. para esto nos vamos a las Configuración de webMAN
      2. y dentro buscamos el Buscar contenidos y en Buscar juegos/videos por LAN y le agregamos la ip de nuestro celular para esto todo tiene que estar dentro de la misma red
    • después le damos a actualizar para refrescar la lista de Juegos
    • y listo a disfrutar
