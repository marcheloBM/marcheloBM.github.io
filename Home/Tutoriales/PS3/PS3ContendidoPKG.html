PS3 Contendido de un .pkg

PkgView es una herramienta para extracción de archivos pkg de PS3. Es una herramienta 100% legal ya que no contiene código de Sony. Además, PkgView tiene una interfaz gráfica con la que simplifica la extracción tanto de archivos sueltos como del paquete completo.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip .rar o .zip

Programas para este Tutorial

  1. RAR PkgView
  2. WEB Oficial github.com

    Para Empezar este Tutorial

  1. Descargamos los archivos y lo pasamos a nuestra USB en la raíz
    • Descomprimimos el archivo .rar o .zip
      1. PkgView_1.3.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • El archivo PkgView.exe (podemos abrirlo como administrador si queremos)
    • Una vez abierto podemos seleccionamos File/Open
    • Podemos seleccionar cualquier archivo .pkg
    • Para ver su contenido
    • Podemos extraer o añadir contenido a nuestro archivo .pkg
