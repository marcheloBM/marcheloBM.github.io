PS3 ISO TOOLS

PS3 ISO TOOLS podemos crear y convertir juegos a iso de PS3.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip
  3. Un Juegos de PS3 en formato carpeta

Programas para este Tutorial

  1. RAR PS3 ISO TOOLS

    Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. PS3 ISO TOOLS V2.2.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Dentro de la carpeta buscamos y ejecutamos PS3_ISO_TOOLS V2.2.exe
    • Una vez abierto tenemos
      1. Create ISO: podemos crear ISO
      2. Extract ISO: podemos extraer el contenido de una ISO
      3. Split ISO: podemos dividir archivos grandes en partes de 4GB para nuestros USB en Fat32
      4. Join ISO: nos permite unir un iso dividido
      5. Modify ISO: podemos modificar una ISO ya creada
      6. Patch ISO: para modificar el ISO para CFW u otro parámetro
    • c. Para crear uno ISO de damos a Create ISO
      1. Las recomendaciones serian como en la imagen
      2. Le damos a continuar y seleccionamos la ruta donde esta nuestro juego en formato carpeta
      3. Después nos pide donde queremos guardar nuestro juego y seleccionamos la ruta
      4. Una vez seleccionada esperamos que termine la conversión de nuestro juego
      5. Una vez terminado la conversión nos mostrar un archivo de texto donde esta toda la información del proceso
      6. Después copiamos el archivo a nuestra USB o HDD de la consola PS3
  3. Y estaríamos listo
