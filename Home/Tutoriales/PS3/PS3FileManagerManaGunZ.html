PS3 FileManager-ManaGunZ

Este gestor de ficheros FileManager nos da la posibilidad de navegar por todos los directorios de la PS3, así como de cualquier memoria externa que tengamos conectada. Podremos copiar y eliminar de la forma más simple imaginable.

ManaGunz es una utilidad para lanzar nuestras ISOs y juegos CFW desde PS3. Incluye payloads de IRIS, Multiman y COBRA, para poder gestionar lo mejor de cada launcher reunido en uno mismo. ManaGunz en nuestro HEN nos permite cargar y gestionar PS3ISO, BDISO, PS2ISO y PSXISO. Es capaz de crear directamente ISO's en nuestra PS3 o de copiar los juegos a CFW.

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32

Programas para este Tutorial

  1. PKGFileManager
  2. PKGManaGunZ
  3. Web OficialFileManager-ManaGunZ

Empezando este Tutorial

  1. Descargamos el archivo lo descomprimimos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos a Administrar archivos PKG
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • Y instalamos nuestra aplicación
    • Una vez instalada nos saldrá en el menú de Juegos

Para Empezar con FileManager

  1. Con esta aplicación podemos gestionar nuestros archivos
  2. Podemos copiar información desde nuestra USB hacia nuestro disco duro de la consola PS3
  3. Podremos sacar información de nuestra consola hacia un disco duro o USB como ejemplo los SaveData de nuestros juegos
  4. Una pequeña ayuda para mandos

Para Empezar con ManaGunZ

  1. Con esta aplicación podemos iniciar nuestros juegos en formato ISO o formato carpeta
  2. Para poder usar esta aplicación tendremos que iniciar HEN
  3. Una vez iniciado la aplicación presionamos para abrir las configuraciones de la aplicación
    • En configuración Global le damos a descargar caratulas de juegos
    • Para poder tener nuestros juegos con sus respectivas caratulas
    • Después nos regresamos a Configuración del Interface y en tipo de interface le colocamos Flow
    • Con las demás podremos ir probando los ajustes
    • Regresamos a Configuración del Tema y le cambiamos Default por Wallshow Grey
    • Y listo lo demás es solo ir probando
  4. Para iniciar nuestros juegos le damos al
  5. Con algunos juegos de PS2 podemos presionar para cambiar algunas configuraciones como ejemplo la resolución y mas
  6. Y estaríamos listo
