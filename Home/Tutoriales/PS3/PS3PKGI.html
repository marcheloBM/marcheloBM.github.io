PS3 PKGI

PKGi PS3 es un port para Playstation 3. Esta aplicación nos permite descargar e instalar archivos .pkg directamente en nuestra Playstation 3. Descargas automáticas. Simplemente selecciona un archivo y automáticamente se descargará al disco duro de la consola (direct mode) o se añadirá a la cola de descargas (background mode) utilizando el Download Manager interno.

Empezando este Tutorial

  1. Descargamos los archivos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos a Administrar archivos PKG
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • Y instalamos estos pkg de nuestra aplicación
      1. PKGi 1.2.?
      2. Config o TXT (Para poder dejarlo activo depende de la versión a instalar)
      3. Auto_DB_PKGi.pkg (para tenerla actualizada la BD)
    • PKG con la última versión disponible
    • Una vez instalada nos saldrá en el menú de RED
  2. Una vez terminado abrimos nuestra aplicación PKGI
    • Nos mostrara una lista con todos los juegos disponibles para instalar
    • Todos los juegos con un punto blanco son los juegos ya instalados en nuestra consola PS3
    • Con el nos mostrara una mínima información
    • Con el sacamos el menú de configuraciones
      1. Podemos buscar por título, región, nombre (A-Z o Z-A) o tamaño
      2. Podemos filtrar el contenido por Todo, Juegos DLCs, Temas, Avatares, demos, etc.
      3. Podemos filtrar por región Asia, Europa, japon y USA
      4. Y en opciones podemos configurar como descargar los juegos Directo o con back del juego, quitar la música y actualizar nuestra aplicación
      5. Y para seleccionar le damos a la y para salir del menú le damos o
    • Para instalar algo
      1. Seleccionamos un juego (si dice RAP not Required no necesita licencia para abrir)
      2. Para comenzar a instalar le damos a la encima del juego
      3. Si seleccionamos Direct DL nos descargar e instalara el juego si seleccionamos Back. DL nos descargar un pkg q después tendremos que instalarlo
      4. Para la opción de Back. DL los archivos PKG están
        • Se encuentran en Administrar Archivos PKG/Contenido de PlayStation Netmork
        • Podemos instalar todos los juegos que se han descargado
        • Con triángulos podemos eliminar o instalar también
      5. Para salir de la aplicación le damos a
  3. Para la BD actualizada
    • Solo nos quedaría instalar Auto_DB_PKGi.pkg o (Config o TXT para versión más antiguas) para poder tener
    • Y estaríamos listo a disfrutar nuestro juego
