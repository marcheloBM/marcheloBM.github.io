PS3 Apollo Save Tool

Apollo Save Tool es una aplicación para administrar archivos de partidas guardadas, trofeos y licencias en PlayStation 3.

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32
  2. Conexión a Internet en nuestra PS3

Para Empezar este Tutorial

  1. Descargamos el archivo lo descomprimimos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos a Administrar archivos PKG
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • Y instalamos nuestra aplicación
    • Una vez instalada nos saldrá en el menú de Juegos
  2. Para comenzar a exportar o importar SaveData
    • En nuestra Computadora colocamos nuestro USB y revisamos que tenga el formato FAT32
    • Dentro de la USB colocamos nuestros SaveData descargados o nuestra copia de seguridad de la siguiente forma
    • Carpetas de datos
      1. PS3
        2. PS3 Carpeta
        USB Saves sus archivos deben ser colocados en /dev_usb00x/PS3/SAVEDATA/ .
        HDD Saves los archivos se escanearán desde /dev_hdd0/home/000000XX/savedata/, dónde XX es el actual User ID.
        HDD Licenses /dev_hdd0/home/000000XX/exdata/ (*.rif)
        USB Licenses /dev_usb00x/exdata/ (*.rap)
      2. PS2
        3. PS2 Carpeta
        USB Saves /dev_usb00x/PS2/SAVEDATA/ (*.xps, *.max, *.psu, *.cbs, *.sps)
        VMC Cards /dev_usb00x/PS2/VMC/ (*.vmc)
        VME cards /dev_usb00x/PS3/EXPORT/PS2SD/
        PSV saves /dev_usb00x/PS3/EXPORT/PSV/
        HDD VME cards /dev_hdd0/home/000000XX/ps2emu2_savedata/ dónde XX es el actual User ID.
        HDD VM2 cards /dev_hdd0/savedata/vmc/
        HDD ISOs /dev_hdd0/PS2ISO/ (*.bin, *.iso, *.bin.enc)
        USB ISOs /dev_usb00x/PS2ISO/ (*.bin, *.iso, *.bin.enc)
      3. PS1
        4. PS1 Carpeta
        USB Saves /dev_usb00x/PS1/SAVEDATA/ (*.mcs, *.psx)
        PSV Saves /dev_usb00x/PS3/EXPORT/PSV/
        HDD VM1 cards /dev_hdd0/savedata/vmc/
    • Una vez listo solo conectamos nuestro USB a nuestra consola PS3 e iniciamos nuestra aplicación
      1. Si vamos a pasar SaveData desde USB a la Consola abrimos USB Saves
      2. Si vamos a pasar SaveData desde la Consola a nuestro USB abrimos HDD Saves

Aplicación (homebrew)

  1. Uso de la Aplicación
    • Usar la aplicación es simple y directo:
    • Mueve / para seleccionar el archivo de partida guardada que deseas parchear y presiona botón . La pantalla del parche mostrará las correcciones disponibles para el archivo. Seleccione los parches y haga clic en Apply.
    • Para ver los detalles del elemento, presione . Se abrirá el menú contextual en la pantalla. Pulse botón para volver a la lista.
    • Para recargar la lista, presione .
    • Presione los botones de activación / o / para mover las páginas hacia arriba o hacia abajo.
  2. Menú
    • Trophies (Gestión de trofeos)
      1. En este menú podemos gestionar nuestros trofeos de nuestros juegos podemos desbloquear o exportar como una copia de seguridad en formato .zip
    • USB Saves (Saves data en USB)
      1. En este menú podemos colocar nuestros saves datas (descargados o copiados con anterioridad) en nuestro USB
    • HDD Saves (Saves data en HDD Interno)
      1. En este menú tendremos acceso a nuestros saves datas de nuestros juegos dentro del disco duro de nuestra consola PS3
    • Online DB (Base de datos Online)
      1. En este menú tendremos una lista con todos los saves datas disponibles para descargar para nuestro juego
    • User Backup (Copias de seguridad del usuario)
      1. Podemos realizarle a nuestra consola copias de seguridad a flash2, Licenses, PS2 .VM2 memory cards a nuestro USB conectado a la consola
    • Settings (Configuraciones)
      1. Podemos cambiar algunas configuraciones de la aplicación como por ejemplo sonido, pantalla y actualizaciones de la aplicación
    • Abolit (Ayuda)
      1. Información de los creadores de esta gran aplicación para nuestras consolas PS3
