PS3 9420 Caratulas

9420 caratulas para nuestros juegos de PS3, PS2 y Retro este pack de caratulas son para nuestros Hombrew de multiMAN y webMAN MOD solo descargar e instalar y listo

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32

Archivos Para descargar

Archivos Servidores
Caratulas Mega
WEB Oficial github.com

    Para Empezar este Tutorial

  1. Descargamos los archivos y lo pasamos a nuestra USB en la raíz
    • Para instalar el .pkg nos vamos a Administrar archivos PKG
    • Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
    • c. Y instalamos estos pkg del pack
      1. EP0001-BLES80608_00-COVERS0000000000.pkg
      2. EP0001-BLES80608_00-COVERS00000RETRO.pkg
      3. RetroArch_Psx-Place_Community_Edition_unofficial_beta-20220315.pkg
    • PKG con la última versión disponible
  2. Una vez terminada la instalación
    • Podemos revisar multiMAN para verificar que estén todas las caratulas de nuestros juegos
    • Para webMAN MOD debemos refrescar la lista de nuestros juegos
      • Para esto nos vamos a Juegos de webMAN/Configuración de webMAN y seleccionamos
        1. Actualizar juegos de webMAN y el XMB: El XMB se volverá a cargar tras actualizar la lista de juegos
        2. Actualizar Juegos de webMAN: La lista de juegos se mostrará la próxima vez que cargue el XMB
        3. Actualizar lista web de juegos: La lista de juegos en formato HTML se actualizará para el modo web
      • Nos quedaría solo reiniciar la consola
      • Y podremos visualizar las caratulas de nuestros juegos
    • Y estaría listo la instalación del pack de caratulas
