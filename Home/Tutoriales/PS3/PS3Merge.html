PS3Merge

PS3Merge es una herramienta que permite a los usuarios fusionar archivos divididos por varios administradores de copia de seguridad de PS3

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

  1. RAR PS3Merge
  2. WEB Oficial karmian.org

    Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. ps3merge-1-0-1-0.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Abrimos el instalador ps3merge-1-0-1-0.exe
    • Una vez terminada la instalación abrimos PS3Merge que se encuentra en nuestro escritorio o menú de inicio
    • Buscamos las partes de nuestro juego PKG para unir
    • Le cambiamos el nombre para que nuestra aplicación lo detecte
    • Le asignamos el nombre.666xx donde las xx serían las partes 01,02,03, etc nos quedaría Beyond Two Souls66601 por ejemplo
    • En nuestra aplicación PS3Merge le damos a Execute y nos pedirá buscar la carpeta donde están todos nuestros archivos modificados
    • Seleccionamos la carpeta y le damos aceptar y esperamos que termine el proceso
    • Una vez terminado la aplicación nos quedara un archivo solo quedaría agregar la extensión .pkg al final
  3. Y listo
