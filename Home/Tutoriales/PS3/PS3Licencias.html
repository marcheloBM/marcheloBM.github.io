PS3 Licencias

Existe el típico pkg de las 15.280 licencias que las instalas y ya deberías de poder licenciar y activar casi todo el contenido sin que salga la bolsita típica de que hay que renovar (comprar)

Para esta Guia

Requisitos para este tutorial

  1. USB en formato FAT32

Programas para este Tutorial

  1. En PKG o ZIP15.280 Licencias

Empezando este Tutorial

  1. Forma Automática de instalación
    • Descargamos el archivo 15.280_Licencias.pkg y lo pasamos a nuestra USB en la raíz
      1. Este tipo de instalación se puede demora mucho en nuestra consola ya que tiene que instalar 15.280 archivos en nuestra consola PS3.
      2. Para instalar el .pkg nos vamos a Administrar archivos PKG
      3. Seleccionamos Instalar archivos PKG/Directorio estándar (USB)
      4. Y instalamos nuestra aplicación
      5. Una vez instalada estaría todas las licencias listas para usar
  2. Forma manual de instalación:
    • Todo juego o dlc tiene un Title ID como, por ejemplo: (BLES01032 Físico / NPEB01032 Digital).
    • Simplemente necesitas buscar el rap con este ID BLES01032 si es en formato carpeta o NPEB01032 si es Digital de PSN y copiarlo en la carpeta
    • Desde el USB en la carpeta: exdata
    • En el disco interno de la consola o FTP: dev_hdd0/exdata
    • Recuerda que si el title ID del rap (licencia) no concuerda con el title ID del juego no funcionará.
  3. Una vez instalado podremos instalar cualquier juego o DLC en nuestra consola.
  4. Solo nos quedaría descargar y disfrutar nuestros juegos.
