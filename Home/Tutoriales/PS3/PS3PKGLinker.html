PS3 PKG Linker

PKG Linker es una herramienta de Windows para cargar imágenes de juegos y otras aplicaciones en una PS3 habilitada para CFW/HAN. Este método permite a los usuarios sortear la limitación de tamaño de archivo de 4 Gb.

Para esta Guia

Requisitos para este tutorial

  1. PC o Notebook con Windows 7, 8, 8.1, 10
  2. Para descomprimir archivos .rar o .zip

Programas para este Tutorial

  1. RAR PKG Linker

    Para Empezar este Tutorial

  1. Descargamos los archivos
    • Descomprimimos el archivo .rar o .zip
      1. PKG_Linker_V2.0_Installer.rar
    • Con la última versión disponible
    • Una vez descargado nuestro archivo
  2. Lo extraemos y ejecutamos
    • Al extraer precaución con el antivirus de nuestro pc puede eliminar archivos del programa
    • Buscamos y ejecutamos PKG Linker V2.0.exe (Si hay problema ejecutarlo como administrador)
    • Pasamos todos los archivos PKG donde esta nuestra aplicación
    • Una vez abierta nuestra aplicación le damos Rescan para que muestre todos los archivos que recién le pasamos
    • Para conectar a nuestra consola vamos a Setting
      1. En el apartado Ping Tool colocamos la IP de nuestra consola
        • Para obtener la IP de nuestra consola vamos a Ajuste/Ajuste del Sistema/información del sistema
      2. Colocamos la IP y le damos a Ping PS3 si nos sale de Success en verde estamos listo, pero si dice ERROR: TIMEOUT en rojo estaría mal puesta la IP
      3. En Server Setup colocamos Create List.pkg for HAN para nuesta consolas con HEN
    • Una vez listo nos vamos a PKG List y buscamos el botón Start Server
      1. Cuando nos muestre una notificación estaría corriendo el programa ya lo podemos minimizar
      2. Dentro del programa nos crea un archivo Package_List.pkg que tenemos que instalar en la consola por USB o FTP como les saga más fácil
      3. Una vez terminada vamos a la tienda de WIKISTORE 4.8? dentro buscamos Herramientas Avanzadas/Instalar Paquetes desde PC y buscamos? Install Packages Webserver
      4. Y dentro estarán todos los archivos PKG que le pasmos a nuestro programa
      5. Solo queda descargarlos a nuestra consola
    • Y listo
