    VitaShell

    VitaShell es una alternativa para PS Vita en forma de aplicación VPK que te permite reemplazar el LiveArea (menú táctil de la consola) por un gestor de archivos a la vieja usanza, pero que soporta muchísimas más opciones. Para poder instalar este homebrew es necesario tener previamente instalado HENKaku.

    VitaDB

    VitaDB es el repositorio de aplicaciones de Vita más utilizado por los propietarios de consolas PS Vita habilitadas para homebrew . La base de datos alberga miles de juegos y software de utilidad, con un sitio web donde las personas pueden descargar manualmente el homebrew para su instalación manual en su consola. También proporciona una API, lo que permite a los desarrolladores acceder a la base de datos mediante programación.

    RETROARCH

    RetroArch es una interfaz para emuladores, motores de juegos y reproductores multimedia. Le permite ejecutar juegos clásicos en una amplia gama de computadoras y consolas a través de su elegante interfaz gráfica. Los ajustes también están unificados, por lo que la configuración se realiza de una vez por todas. ¡RetroArch tiene funciones avanzadas como sombreadores, netplay, rebobinado, tiempos de respuesta del siguiente cuadro, runahead, traducción automática, funciones de accesibilidad ciega y más!

    YOYO LOADER

    YoYo Loader es un cargador para libyoyo.so, la aplicación oficial de GameMaker Studio Runner para Android, para PS Vita. YoYo Loader funciona cargando dicho ejecutable ARMv7 en la memoria, resolviendo sus importaciones con funciones nativas y parchándolo para que funcione correctamente. Al hacerlo, es básicamente como si emulamos un entorno minimalista de Android en el que ejecutamos de forma nativa el ejecutable tal como está. Esto permite ejecutar potencialmente cualquier juego creado con GameMaker Studio.
