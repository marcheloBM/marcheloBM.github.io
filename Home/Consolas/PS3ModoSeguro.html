PS3 Modo Seguro

El procedimiento para iniciar PlayStation 3 en modo seguro es el siguiente:

  • apague la consola y asegúrese de que esté encendida (LED rojo fijo)
  • presione el botón de encendido y manténgalo presionado sin quitar el dedo. Sonará un primer "bip" después de cinco segundos, seguido de un segundo "bip" después de diez segundos. La consola se apagará, por lo que podrá retirar el dedo. El led rojo estará fijo
  • presione el botón de encendido nuevamente. Después de cinco segundos, se emitirá un primer "pitido", seguido después de unos segundos por dos "pitidos" consecutivos. Retire su dedo inmediatamente
  • Si ha seguido el procedimiento correctamente, la pantalla debería activarse pidiéndole que presione el botón PS en el controlador de PlayStation 3

Menu MODO SEGURO PS3

  • Esta pantalla aparecerá:
  • Una vez que el controlador está conectado, aparecerá este menú:
  • Brevemente, esta es la descripción punto por punto:

  1. Reinicia el sistema PS Vita: Simplemente reinicia la consola.
  2. Reconstruye la base de datos: Muy útil si al desinstalar alguna aplicación se queda un icono en blanco, alguna aplicación falla... Corrompe la base de datos actual de la consola y de esta forma la consola debe reconstruirla para funcionar, las listas, los mensajes, los iconos se reorganizan de nuevo.
  3. Formatear tarjeta de memoria: Muy simple, formatea la tarjeta de memoria de PS Vita y la deja como nueva, muy útil si vais a vender la tarjeta y no queréis dejar rastro, aunque también puede hacerse desde los ajustes de la consola. Tiene mas utilidad si por ejemplo instaláis un programa corrupto y al iniciar el menú con la tarjeta instalada en la consola esta falla.
  4. Restaurar el sistema PS Vita: Deja la PS Vita como cuando la comprastes, pero con el firmware actual, perfecto si vas a vender tu PlayStation Vita y quieres que el nuevo propietario la configure de nuevo.
  5. Actualizar el software del sistema: Simplemente actualiza el firmware de la consola a traves de un PC, Mac, una consola PlayStation 3, o desde una tarjeta de juego de PS Vita. Conectas la consola mediante USB y con el gestor de contenidos en PC o Mac o bien con una PlayStation 3 con firmware 4.00 o superior.
  6. Actualizacion del sistema: le permite actualizar el software del sistema PlayStation 3, que solo se puede ejecutar a través de un medio de almacenamiento.

En concreto las opciones son:

  • Actualizar mediante una conexión a un sistema PS3 (TM)
  • Actualizar mediante una conexión a un PC
  • Actualizar desde la tarjeta PS Vita

Espero que esta pequeña guía ayude a solucionar pequeños problemas que vayan surgiendo.
